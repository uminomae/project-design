# What Is Value — Survey Note

This note is a cross-disciplinary survey of academic literature on the definition, polysemy, and Japanese–English translation gaps surrounding "value," conducted using LLMs (Claude / GPT) and compiled and verified by pjdhiro. It is not an original theory by pjdhiro but an organized review of publicly available literature.

---

## Summary

"Value" is a polysemous term bundling at least five distinct problem domains: price, utility, norms, meaning, and the symbolic. Japanese 価値 (kachi) branches into the English terms value / values / worth / utility / significance / merit / price, making cross-domain confusion common in translation. Value is neither something fixed inside an object nor merely a subjective feeling; it is a multi-layered concept that arises within the relations among object, subject, institution, and context.

## Five Problem Domains

| Problem domain | Definition | Representative theory | English equivalent |
|---|---|---|---|
| Price | Monetary expression in the exchange of goods and services | Classical economics (Smith, Ricardo), Marx's labor theory of value | price, market value |
| Utility | Degree of goal achievement or desire satisfaction | Marginal utility theory, Prospect theory (Kahneman & Tversky) | utility, usefulness |
| Norms | Beliefs and standards concerning good/bad, right/wrong | Axiology (Moore, Ross), Schwartz Basic Human Values | values, moral value |
| Meaning | Importance, significance, weight as lived experience | Phenomenological approaches, meaning-of-life literature | significance, meaning |
| Symbolic | Value as status, honor, prestige, or sign | Bourdieu's symbolic capital, Baudrillard's sign value | prestige, symbolic value |

## Japanese–English Translation Gaps

| English | Primary Japanese equivalent | Point of divergence |
|---|---|---|
| value | 価値 | Japanese 価値 is an umbrella term covering the scope of all terms below |
| values | 価値観・信条 | Easily conflated with singular value (importance) |
| worth | 値打ち・自己価値 | Strong nuance of "being worth it" |
| utility | 有用性・効用 | In economics, a representation of preference ordering, not psychological pleasure |
| significance | 意義・重要性 | Straddles "meaning" and "importance" |
| merit | 長所・功績 | Japanese メリット leans toward gain, but English merit includes legitimacy |

The translation of "価値がある" (has value) diverges by context: if expensive → valuable; if cost-effective → worth it; if instrumental → useful; if deeply meaningful → meaningful.

## Major Theories

| Theory | Author | Key point |
|---|---|---|
| Basic Human Values | Schwartz (1992) | Circular structure of 10 basic values. Oppositions and compatibilities described in 4 higher-order dimensions |
| Four Forms of Capital | Bourdieu (1986) | Economic / cultural / social / symbolic. Inter-capital conversion and reproduction |
| Anthropological Value Theory | Graeber (2001) | Integration of three traditions: sociological, economic, and linguistic |
| Orders of Justification | Boltanski & Thévenot (1991) | Six orders of justification. Describes value conflicts as grammar |
| Prospect Theory | Kahneman & Tversky (1979) | Reference-point dependence, loss aversion, probability weighting |
| Active Inference | Friston et al. | Dual structure of extrinsic value and epistemic value |
| Value Sensitive Design | Friedman et al. (2006) | Three-phase investigation for embedding human values in technology design |
| Use value / Exchange value | Smith, Marx | Distinction between usefulness of goods and exchange ratio |

## Cross-Domain Issues

| Issue | Content |
|---|---|
| Objective vs subjective | Is value a property of the world, or a relation between evaluator and object? Relational views are increasing |
| Discovery vs attribution | Is one finding value that "is there," or is value humanly conferred? |
| Value vs price | Price is one part of the monetary expression of value. In art, dignity, and nature, value resists pricing |
| Value vs meaning | Value concerns "why it matters"; meaning concerns "what it represents and how it is experienced" |
| Evaluation vs value | Value is the worth of the object; evaluation is the act of estimating it |
| Commensurability | Can heterogeneous values be placed on a single scale? Resistance to commensuration is strong in domains of the sacred and human dignity |
