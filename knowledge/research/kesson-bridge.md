## 1. 価値 → D2（欠損）・D4（情動の構成）

### 接続

| 欠損駆動思考 | 価値レポートの知見 | 接続の内容 |
|---|---|---|
| D2 5類型「正当化欠損」 | 価値の5問題系「規範」 | 行動と価値観のズレは、規範的価値の問題系そのもの。「何に照らしてズレと感じるか」は価値体系に依存する |
| D2 5類型「意味欠損」 | 価値の5問題系「意味」 | 意味の喪失・空虚は、現象学的「生きられた重要性」の崩壊。D2の1人称記述と直結 |
| D4 生存-信頼座標系 | 5問題系（価格/効用/規範/意味/象徴） | D4は情動生成メカニズムであり価値分類論ではない。5問題系はD4への入力（どの問題領域で欠損が生じたか）として接続される。写像経路は part-4 4.5.3節に部分的根拠あり |
| D2「主観的経験」 | 「生きられた重要性と主観価値の関係」（未解決問い） | 認知科学の subjective value と D2 の1人称記述は同じ現象の異なる記述水準か？ |

### 補強ポイント

- **D2の5類型に価値の問題系を対応させることで、欠損の「内容」をより精密に記述できる可能性がある**。現在のD2は「何がズレたか」の形式的分類であり、「何の価値に照らしてズレたか」は未記述
- **[S] D4の生存-信頼座標系と象徴問題系の関係は保持論点**。象徴（地位・名誉）が信頼軸（承認欠如=孤立）に吸収可能か、第三の評価軸を要するかは未決。D4は情動生成メカニズムであり価値分類論ではないため、問いは「象徴的欠損が生存-信頼軸にどう入力されるか」として定式化する

### 浮上する問い

1. 欠損の5類型と価値の5問題系は独立なのか、交差するのか
2. 内在的価値（intrinsic value）は、欠損の検出閾値に影響するか——「それ自体で価値がある」と見なすものの欠損は、より低い閾値で検出されるのか
3. 価値の比較可能性問題（commensurability）は、複数の欠損が同時に生じたときの優先順位問題と同型か

## 2. 信頼 → D3（抱持）・D1（欠損駆動思考）

### 接続

| 欠損駆動思考 | 信頼レポートの知見 | 接続の内容 |
|---|---|---|
| D3 成立条件 L0-L1「安全な関係（E10）」 | 信頼の共通核「脆弱性」 | 欠損を抱持するには脆弱でいられる関係（信頼）が必要。信頼なき場では欠損は即座に防衛反応に変わる |
| D3 成立条件の4層 | 山岸の安心/信頼区別 | **共鳴**（分析軸が異なるため直接対応ではない）。山岸の安心（制度的リスク除去）はD3のL1-L2「外部Container」に機能的に近い。山岸の信頼（不確実性下の能動的委託）はD3のL2-L3（epoche・ネガティブ・ケイパビリティ）に共鳴する。ただし山岸は信頼の認識論的性格を、D3は抱持の成立条件の重みを記述しており、記述対象が異なる |
| D3 循環（L3→L1-L2再入力） | 信頼の修復メカニズム | 修復は「もう一度警戒を解けるか」の閾値問題。抱持の循環も「問いの意味が更新される」循環。構造的類似がある |
| D1「問いとして拾う態度」 | epistemic trust | **堅牢**。epistemic trust（他者の証言を受容可能にする前提）は、D1的態度を可能にする認識論的条件の外在化例。外部からの誤差信号を「問い」として拾うにはepistemic trustが作動する必要がある（ただしD1は孤立した観察者でも成立しうるため「必要条件」ではなく「条件の一つ」） |

### 補強ポイント

- **D3の成立条件と信頼研究の語彙は共鳴する**（直接対応ではない）。山岸の安心（制度的リスク除去）とD3のL1-L2（外部Container）、山岸の信頼（不確実性下の委託）とD3のL2-L3（認知的枠組み）は、分析軸が異なる（信頼の認識論的性格 vs 抱持の成立条件の重み）が、「外部構造が不確実性への対処を支える」点で機能的に共鳴する
- **「信頼は状態か特性か関係か過程か」という未解決問いは、D3（抱持）にもそのまま当てはまる**。抱持を状態として記述するか、過程として記述するかで理論の射程が変わる

### 浮上する問い

1. 抱持は信頼の一形態か、それとも信頼を前提とする別の機能か
2. 信頼の破壊は抱持の崩壊を引き起こすか——関係が壊れたとき、欠損は問いではなく脅威になる転換点はどこか
3. 自己信頼（self-trust）と抱持の関係——D3の成立条件に「自分を信頼できること」は含まれるか

## 3. デザイン思考 → D1（欠損駆動思考）・D3（抱持）

### 接続

| 欠損駆動思考 | DT レポートの知見 | 接続の内容 |
|---|---|---|
| D1「棄却される誤差を問いとして拾う」 | abduction の設計固有性 | 科学の abduction が既存世界の説明仮説を立てるのに対し、design の abduction は「まだ存在しないが望ましいもの」を扱う。D1 は後者に近い——棄却された誤差から「あるべきだが、ない」を読み取る |
| D1 の態度 | framing / frame creation | **部分的**。何を「欠損」と見るかは frame に依存する。Dorst の frame creation（価値判断・境界設定・評価軸の設計を含み外化を伴う操作）は、D1 の「問題設定の組み替え」と部分的に類似するが、D1 は態度（L2）であり frame creation は外化を伴う操作であるため、統合は保持論点 |
| D3 循環（L3→L1-L2再入力） | Schon の reflection-in-action | **共鳴**。両者は「反復的に更新するループ」を持つ点で構造的に類似する。ただし R-i-A は外部環境との対話（状況の back-talk）を主語とし、D3 層間再入力は内部の意味更新循環を主語とする。主語の差は本質的であり、操作的同一視はできない |
| D1 の否定形 | 形骸化 | **堅牢**。形骸化は D1 反転として2レベルで記述できる: (a) 個人態度レベル — 誤差を問いとして拾わず棄却する姿勢、(b) 組織構造レベル — D3-a L1-L2「外部Container」の欠如（評価制度・権限配分が固定され、組織として抱持が成立しない）。innovation theater（パターン5）は主に (b) |

### 補強ポイント

- **D1 は abduction の前態度として位置づけられる**。D1（棄却される誤差を問いとして拾う態度）が先に作動し、その後に abduction（仮説生成）が続く。D1 は abduction そのものではなく、abduction を可能にする構えである
- **frame creation と D1 は部分的に類似する（統合は保持論点）**。frame creation は外化を伴う枠組み再構成操作であり、D1 は態度（L2）。「問題設定の組み替え」レベルで重なるが、操作と態度の層の違いがある
- **形骸化は D1 反転の2レベルとして記述できる**: (a) 個人態度レベル（誤差を棄却する姿勢）と (b) 組織構造レベル（D3-a L1-L2 外部Container の欠如）。DT 批判を欠損駆動思考の防衛条件の議論に転換でき、介入戦略を個人変容と構造変容に分離して設計できる

### 浮上する問い

1. D1 の「拾う」は abduction か、それとも abduction の前段階（問いの発見）か
2. wicked problems は欠損の5類型が束になった「複合欠損」として記述できるか——定義・正解・最適解を拒む問題は、単一類型では捉えられない欠損か
3. 形骸化防止の制度的条件は、D3 の成立条件（L1「外部Container」）と同じ議論か

## 補足A: Dorst 2011 frame creation — D1 との接続精査

Dorst 2011 "The Core of 'Design Thinking' and its Application" の精査結果。

### frame creation の操作的定義

**フレーミングの論理構造（IF-THEN）**:
> IF we look at the problem situation from this viewpoint, and adopt the working principle associated with that position, THEN we will create the value we are striving for.

**推論方程式**: WHAT（thing）+ HOW（working principles）→ VALUE

### Abduction の2種類（Dorst 2011 / Roozenburg 1993）

| 種類 | 既知 | 未知 | 推論形式 |
|------|------|------|----------|
| Abduction-1（explanatory） | WHAT, HOW | VALUE | WHAT + HOW → ?（VALUE を見つける） |
| Abduction-2（design固有） | VALUE | WHAT, HOW | ? + ? → VALUE（WHAT と HOW を同時に構想する） |

Abduction-2 が frame creation を要求する理由: WHAT も HOW も未確定な状況では、問題の枠組みそのものを構築（frame creation）しなければ解候補の空間が定まらない。

### D1 との層の差（精密化）

| 比較軸 | frame creation（Dorst） | D1（欠損駆動思考） |
|--------|------------------------|-------------------|
| 性格 | 操作（deliberate strategy / key practice） | 態度（L2） |
| 外化の要否 | 構成要素（プロトタイプ等で試す） | 不要（態度レベルで成立） |
| 時間スケール | プロジェクト単位の反復探索 | 個々の認知エピソード（棄却の瞬間） |

**結論**: D1 は frame creation の先行条件の一つ（ただし必要条件ではない）。D1 が作動して「棄却されるはずの誤差を拾う」ことで Abduction-2 の状況に向き合う構えが生まれやすくなり、その後に frame creation（viewpoint + working principle の仮定的採用）が展開する。ただし、問題設定が外部から与えられる場合には D1 なしでも frame creation は生じうる。

## 補足B: 価値5問題系 × D2の5類型 — 全25組対応強度

| | 観測欠損 | 主体欠損 | 正当化欠損 | 一貫性欠損 | 意味欠損 |
|---|---|---|---|---|---|
| **価格** | **高**（文脈依存） — 価格予測のズレが個人の財務判断に直結し「欠け」として主観的に経験される場合に高対応。3人称的な市場予測誤差のみでは1人称記述水準との媒介が必要 | 中 — 「自分はこの価格に値するか」という自己評価のズレ | 低 — 価格は制度で決まるため正当化欠損との接続は迂回路 | 低 — 価格と信念矛盾は間接的 | なし — 価格は意味喪失と構造的に接続しない |
| **効用** | **高** — 期待効用 vs 実現効用のズレは観測欠損の直接例 | 中 — 自分の欲求充足の見積もり違い | 中 — 効用と価値観の不一致 | 中 — 複数目標間のトレードオフが信念矛盾に転化 | 低 — 効用消失は意味喪失と異なる構造 |
| **規範** | 低 — 規範と観測事実のズレは認識論的媒介が必要 | 中 — 道徳的自己像と現実行動のズレ | **高** — 行動と価値観のズレは規範問題系の核心 | **高** — 複数規範間の衝突は一貫性欠損の典型 | 中 — 規範の根拠喪失は意味欠損に接続するが論理的ステップが必要 |
| **意味** | 低 — 事実予測のズレから意味論的問いへは飛躍が大きい | **高** — 「自分が何者か」のズレは意味論的問いと直結 | 中 — 行動の意味喪失は正当化欠損を経由 | 中 — 信念矛盾が「何が重要か」という問いを浮上させる | **高** — 意味の喪失・空虚は定義レベルで一致 |
| **象徴** | 低 — 承認の予測外れ（注目されると思ったのに無視された等）は象徴×観測の経路だが、制度的媒介を要するため直接性が低い | **高** — 「この地位を持つ自分」という自己像と現実の社会的位置づけのズレは、主体欠損の定義「自己像と現実のズレ」とほぼ同型 | 中 — 行動が象徴的規範に反するとき | 低 — 象徴体系の内部矛盾はD2定義と粒度が合わない | 中 — 脱象徴化は意味欠損に近い経験を生む |

### 分布パターン

- **高対応**: 7組（価格×観測[文脈依存]、効用×観測、規範×正当化、規範×一貫性、意味×主体、意味×意味、象徴×主体）
- **中対応**: 11組
- **低対応**: 6組
- **なし**: 1組（価格×意味）

判定基準: 高=定義レベルで直結、中=一ステップの論理的媒介を要する、低=間接的・迂回路が多い、なし=接続のコストが理論的記述として正当化できない。

高対応は「計量的問題系（価格/効用）× 観測欠損」と「内部構造系（規範/意味）× 正当化/一貫性/意味欠損」と「象徴×主体欠損」の三群。
