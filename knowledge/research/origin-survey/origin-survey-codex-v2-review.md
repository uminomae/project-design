# origin-survey-codex-v2 吟味レポート

Issue: #46  
対象: `origin-survey-codex-v2.md`  
吟味実施: 2026-03-29  
吟味者: Codex agent team

## 総合評価

B（軽微〜中程度の補強で使える）

v1 で最大の問題だった「反証が消える」「哲学・宗教の固有性が潰れる」は明確に改善した。D13 と D24 の分岐、D16・D18・D20 での否定側の回収、C〜F での慎重な評価は、v2 設計変更が機能したことを示している。一方で、Stage 1 の一部は手動補完であり、`未確認` URL や分野間の出典の厚みの差が残るため、公開版・正本として使う前には補強が必要である。

## 軸1: 事実精度

### 全体所見

v1 より直接URL率は上がり、検索リンク依存は大きく減った。概念対応づけも D13・D24・D18・D20 を中心に精密化されている。ただし、いくつかの領域では「概念は妥当だが出典の直接性が弱い」「隣接分野の文献で代用している」という留保が残る。

### 主な要修正・要確認

| 領域 | 判定 | 指摘 |
|------|------|------|
| D19 文学・文芸学 | 要確認 | `ellipsis` と `silence` の直接URLが未確認で、negative capability 以外の裏取りが薄い |
| D20 法学・政治学 | 要確認 | `nullity / void` の直接URLが未確認。state of nature は十分だが、無効概念の裏づけを補いたい |
| D24 宗教学・神秘主義 | 要確認 | 仏教の `空` の内容自体は妥当だが、個別記事URL未確認のため一次性が弱い |
| D28 舞台芸術 | 要修正 | Google Books の書誌URLは便利だが、可能なら著作本文・出版社・公式紹介ページで補強したい |
| D30 伝統知・技芸 | 要確認 | apprenticeship と Zen で下支えしているが、`初心` や `守破離` に対する直接出典を追加したい |
| D15 美学 | 軽微注意 | `empty space / silence / absence` は知覚哲学寄りで、美学の標準概念としての位置づけに注記があるとより安全 |

### 良い点

- D13 はパルメニデス / ハイデガー / サルトル / `空` を分け、v1 の最大の単純化を回避できている。
- D18 の anomie、D20 の state of nature は、v1 より否定側の基礎文脈を正しく前面に置けている。
- D03、D05、D21、D22 は「存在論的ゼロ」でなく「参照点」として整理されており、概念対応づけが無理になっていない。

### 軽微な注意

- D08、D09、D10 は概念選択は妥当だが、生命・医学系は今後レビュー論文をもう 1 本ずつ追加すると安定する。
- D29 は出典自体はあるが、「criticality = origin」という対応づけの妥当性は軸3で再検討が必要。

## 軸2: 構造的偏り

### 全体所見

v1 に比べると、足場寄りへの回収は大きく改善した。少なくとも D13 パルメニデス、D16、D18、D20 では、否定的扱いが前景化されており、「不足」0個という事態は解消している。また、D13 と D24 の内部多様性も保持された。ただし、横断比較表が 30/30 で「否定あり / 肯定あり」になっているため、依然として「広く解釈すれば両方になる」圧力は残る。

### 偏りが出た領域

| 領域 | 残る偏り | コメント |
|------|---------|----------|
| D03 化学 | 足場寄り | ゼロ基準の「否定側」は参照点の人工性であり、病理的欠如ではない。両義性は弱め |
| D05 地球科学 | 足場寄り | データム不在の不便さはあるが、「負の扱い」は方法論的制約にとどまる |
| D11 薬学 | 足場寄り | placebo は方法論的には強いが、薬学の原点概念としてはやや操作的 |
| D16 歴史学 | 両方化の圧力 | 史料の沈黙は主として欠如であり、肯定側は方法論的転用として限定したほうが安全 |
| D29 複雑系科学 | 対応づけの圧力 | criticality は「原点」より「境界」に近く、survey の都合でゼロ類似概念に寄せている可能性がある |

### v1 から改善した点

- D13 と D24 が一語化されず、内部で複数概念に分かれた。
- D18 と D20 は「病理 / 無秩序」側を先に明示できている。
- Section F が「単一収束」と言い切らず、複数群に分けて慎重化されている。

### 残る歪み

- 出力フォーマット上、「否定的扱い」と「肯定的扱い」を両方書くため、肯定側が薄い領域でも何らかの二次的効用を書き込みやすい。
- 横断比較表は関係類型の細かな違いを圧縮しており、`同時的 / 段階的 / 学派依存 / 無関係` の差がさらに見えにくい。

## 軸3: 三角測量としての有効性

### 全体所見

v2 の F セクションは、v1 より三角測量として健全である。理由は、共通構造を「単一の原点」ではなく、複数の型に分けて扱っているからである。特に「基準点型 / 生成的空白型 / 否定的開示型 / 病理・破綻型」という整理は、v1 の見かけの収束をかなり弱めている。

### 有効だった点

- D01, D03, D05, D21, D22 を「参照面」としてまとめたことで、存在論的な無と測定上のゼロを混同しにくくなった。
- D02, D04, D09, D25, D28, D30 を「生成的空白」として見る整理は、領域横断の比較として一定の説得力がある。
- D13, D19, D24 を「否定が開示条件になる群」として置いたことで、人文系固有のゼロ概念が潰れずに比較できている。

### 収束を弱める点

- D29 は「境界」であり、他群との重なりが大きい。
- D11 placebo や D16 archival silence は、共通構造に含めるときの抽象化コストが高い。
- 4群は v1 の 4 パターンより良いが、なお「基準点」と「境界装置」は部分的に重なるため、完全に独立ではない。

### 次段の具体的問い

1. 「原点」「基準点」「境界」「病理」を分ける判定基準を先に定義した上で、30領域を再分類するとどうなるか。
2. 各領域で、否定側から肯定側へ転化する条件は `同時的 / 段階的 / 学派依存` のどれか。
3. D29 の criticality のような「ゼロではなく境界」の概念を、この調査に含めるべきか。
4. D13 と D24 で、思想・宗派ごとの一次資料を固定したとき、なお同じ群に置けるのか。

## 良い点

- v1 の教訓がそのまま設計変更に反映され、しかも成果物にも現れている。
- 各領域のゼロ概念が「存在論」「方法論」「制度論」「実践論」で異なることが見えるようになった。
- Stage 1 と Stage 2 を分離したことで、仮説への距離感が明示され、反証データを置きやすくなった。
