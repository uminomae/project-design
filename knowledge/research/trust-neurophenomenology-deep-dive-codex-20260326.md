# 信頼の神経現象学：lived experience と第三者データの橋渡し

Issue: #10  
Parent: #6  
作成日: 2026-03-26  
作成: Codex

## 0. エグゼクティブサマリー

信頼研究における神経現象学の位置は、現時点では「確立領域」ではなく「必要性の高い未整備領域」である。心理学、神経科学、経済学、HCI は信頼を質問紙、ゲーム、脳活動、生理反応で多面的に測ってきたが、「信頼している感じ」そのものの経験構造は粗くしか扱われていない。神経現象学はこの空白に対し、第一人称の詳細な経験記述と第三人称の行動・生理・神経データを循環的に結びつける方法論として有望である。

ただし、2026-03-26 時点で trust そのものを正面から主題化した神経現象学研究はかなり少ない。したがって、この領域で言えることは三層に分ける必要がある。第一に、方法論として確立しているもの。Varela の neurophenomenology、Petitmengin の second-person interview、micro-phenomenology、second-person neuroscience はここに属する。第二に、近接領域で相当強く言えるもの。愛着、安全感、社会的相互行為、リアルタイム共同注意、対人予測、身体化された警戒低下などである。第三に、信頼に固有の lived structure についての主張で、これはなお近接研究からの推定が多い。

神経現象学的に信頼を扱うなら、信頼は単一の心的状態ではなく、少なくとも「警戒の緩み」「委譲感」「相手の不透明さへの耐性」「自己開示の可逆性」「身体の防御トーンの変化」「時間的厚み」「裏切り可能性の保持」という経験次元の組として記述されるべきである。重要なのは、これらを事後の一般意見として聴くのではなく、特定エピソードに anchored した経験記述として取ることである。

この視点から見ると、trust game や survey は有用だが不十分である。送金額や自己報告は、trust の有無よりも、戦略、risk tolerance、規範順守、相手の reputation 推定を反映しているかもしれない。神経画像も同様で、信頼の「脳部位」を探すのではなく、予測、価値、脅威調整、身体調整、自己他者境界の変化がどのタイミングで生じるかを見る方が妥当である。

## 1. 何を「信頼経験」と呼ぶのか

### 1.1 最低限の構成要素

- 相手または状況に対して、完全監視や完全制御を一時的に手放すこと
- その手放しが、単なる諦めではなく、何らかの肯定的または可受容な期待を伴うこと
- 裏切りや失敗の可能性がゼロではないこと

### 1.2 候補となる経験次元

- **警戒のトーン**: 相手を見張り続けているか、監視がほどけるか
- **委譲感**: 自分で全てを握らず、相手に一部を渡した感じ
- **可逆性**: 「まだ引き返せる」と「もう賭けた」の境界
- **身体局在**: 胸・腹・喉・呼吸・筋緊張・視線の変化
- **時間性**: 瞬間的閃きか、反復の堆積か
- **自己感**: 露出、拡張、支えられ、脆弱化のどれとして起こるか
- **他者感**: 相手が opaque なままでも近づけるか

### 1.3 信頼は感情か

- 神経現象学的には、信頼を単一感情とみなすより、感情・期待・身体調整・相互行為姿勢の編成としてみる方が有利である。
- 安心、安堵、好意は信頼経験にしばしば含まれるが、信頼それ自体と同一ではない。

## 2. 方法論的基盤

### 2.1 neurophenomenology

- Varela は、第一人称データを第三人称データの「ノイズ」ではなく、相互制約的に用いる必要を主張した。
- この枠組みは trust 研究に直接適用された例は少ないが、信頼の経験構造を細かく分類し、その分類で行動・生理・神経データを再解釈する設計に向いている。

### 2.2 second-person interview / micro-phenomenology

- Petitmengin 系の方法は、特定エピソードを evocation させ、何を感じたかではなく「その瞬間どう経験が展開したか」を取る。
- 信頼研究で有効なのは、一般論として「私は人を信じます／信じません」を聞くのでなく、実際に身を任せた、裏切られた、再び任せられた瞬間を記述させる点である。

### 2.3 second-person neuroscience

- Redcay と Schilbach 以降、社会神経科学では passive observation でなく、相互行為そのものを対象にする必要が強調されている。
- trust は本質的に dyadic / interactional であることが多いため、この転換は重要である。

## 3. 心理学・愛着・社会神経科学との接続

### 3.1 愛着と安全感

- 愛着研究は trust の経験基盤として「応答的他者がいる」という身体化された予測可能性を与える。
- 神経現象学的には、これを threat monitoring のベースライン、自己開示の許容量、支援要請時の身体感覚として記述できる可能性がある。

### 3.2 社会認知

- 心の理論、評判推定、相手の意図推定は trust に重要だが、それだけでは lived trust を尽くさない。
- 同じ相手意図推定でも、「読めないが委ねられる」と「読めないから委ねられない」は経験的に異なる。

### 3.3 社会神経科学

- trust game 系研究は、返報期待、prediction error、reputation updating を可視化してきた。
- しかし、そこでは trust の経験内容は通常 coarse-grained である。
- 神経現象学的アプローチは、たとえば同じ high trust behavior を示した試行でも、どの経験タイプに属するかを分けることができる。

## 4. 1人称と3人称をどう結ぶか

### 4.1 基本設計

1. dyadic cooperation task または自然場面に近い相互行為を行う  
2. 試行ごとに短い experience sampling を入れる  
3. 重要試行を選んで micro-phenomenological interview を行う  
4. 行動・視線・表情・HRV・EDA・EEG/fMRI と照合する  
5. 経験カテゴリを時系列モデルに戻す  

### 4.2 何をマッピングするか

- prediction error と「裏切られた感じ」は一致するか
- HRV の変化と「警戒がほどけた感じ」は対応するか
- eye gaze の安定性と「監視をやめられた感じ」は対応するか
- 相互同調と「身を預けられる感じ」はどの程度結びつくか

### 4.3 注意点

- 脳活動や生理反応は trust そのものではなく、構成要素過程の指標である
- interview の妥当性は取れるが、想起バイアスや narrative reconstruction を避ける工夫が要る
- 研究状況を考えると、まずは exploratory taxonomy を作る段階である

## 5. 候補となる研究プログラム

### 5.1 trust repair の神経現象学

- 裏切り前、裏切り直後、修復後で、経験構造がどう変わるか
- とくに「相手を見る感じ」「身体の硬さ」「撤退衝動」を追う

### 5.2 attachment-informed trust experience

- secure / anxious / avoidant 傾向で、委譲感や警戒の解け方がどう異なるか
- self-report だけでなく、相互行為時の微細経験を比較する

### 5.3 human-AI trust の lived comparison

- 人間相手と AI 相手で、「信頼している感じ」がどこまで同型かを比較する
- ここでは moral vulnerability の有無が経験差になる可能性がある

## 6. 主要論争

1. trust の lived structure は本当に安定的に抽出できるか。  
2. phenomenological interview は reportability を超える経験にどこまで迫れるか。  
3. trust は独立した経験類型か、それとも安全感・予測・好意の束か。  
4. 第三者データとの対応は explanatory か merely correlational か。  
5. 現象学的精密化は ecological validity を上げるのか、逆に実験統制を崩すのか。

## 8. 暫定結論

- **確立していること**: 神経現象学的手法は存在し、相互行為の second-person 研究も成熟しつつある。  
- **近接研究から強く言えること**: trust は予測・安全感・身体調整・相互行為の複合過程である。  
- **まだ薄いこと**: trust 固有の経験類型を体系的に示した神経現象学研究。  

この意味で、信頼の神経現象学は「文献がない」のではなく、「橋はあるがまだ渡り切られていない」段階にある。
