# 信頼という感情：学際的レビューと見取り図

## エグゼクティブサマリー

「信頼」は日常語ではしばしば温かい肯定感・安心と結びつき、「信頼という感情」とも表現される。しかし学術的には、信頼を（i）感情（情動的な結びつき・安心の体感）、（ii）態度（他者や制度に向けた“構え”）、（iii）期待・信念（相手がこう振る舞うだろうという見込み）、（iv）関係の性質（相互行為に埋め込まれた安定性）、（v）制度・社会的メカニズム（不確実性や複雑性を下げる装置）、（vi）行動戦略（協力・リスク選好を含む意思決定）として、分野ごとに異なる単位で扱う。哲学では「信頼は単なる reliance（依拠）と同じではない」「脆弱性（betrayal への vulnerability）を引き受ける態度が重要」といった条件整理が核になりやすい。citeturn0search0turn0search4 一方、組織論では「信頼＝相手の行為に対して脆弱になろうとする意志（willingness to be vulnerable）」という定義が強い影響力を持ち、信頼それ自体（態度）と、信頼行動（リスクを伴う行為）を区別する枠組みが明確に書かれている。citeturn14search5turn1search0

このレビューは、信頼を「感情」として狭く固定せず、（A）概念整理（trust / reliance / confidence / faith、mistrust / distrust / trustworthiness などの区別）、（B）水準（個人内・対人・一般的信頼・制度的信頼・政治的信頼・技術/AIへの信頼）、（C）発達・形成（愛着・裏切り・修復・学習）、（D）測定（質問紙・行動課題・trust game・神経画像・ホルモン・縦断・エスノグラフィー・現象学的面接）、（E）神経現象学（1人称経験と3人称データの相互拘束）、（F）学問間のズレ（同名概念が別物を指す問題）を、横断比較できる「地図」として提示する。神経科学／神経経済学では trust game（投資ゲーム）が代表的プローブとなり、行動・神経反応（線条体など）との結びつきが研究されてきた一方、ゲーム指標が「一般的信頼」や「感情としての信頼」をどこまで代理できるか（外的妥当性、同一個人内安定性）は継続的に検証課題である。citeturn2search5turn13search6turn13search3turn3search4 ホルモン（オキシトシン）研究は、初期の有名研究が示した「信頼増加」効果と、その後の登録追試での不一致が知られ、条件依存性や出版バイアスを含む再評価が必要である。citeturn1search3turn13search1turn13search12

「信頼という感情」を学際的に扱う実務的結論は、信頼を単一量で測るよりも、(1) 対象（誰/何を信頼するか）、(2) 文脈（リスク・依存・権力・監視・規範）、(3) 時間（瞬間状態/関係史/制度経験）、(4) 成分（認知的・情動的・規範的・身体感覚）を分解し、複数手法で照合（triangulation）することにある。とくに神経現象学は、信頼の“感じ”（警戒の緩み、身を預けられる感覚、相手の善意への確信、身体的弛緩など）を高解像度で記述し、同時記録した行動・生理・神経データと相互に制約し合う設計を可能にする方法論として位置づけられる。citeturn0search1turn11search0turn3search6turn11search13

**愛着理論（attachment theory）**  
愛着理論は、乳幼児が養育者を「安全基地（secure base）」として利用できるという確信・期待が、情動調整や対人関係の基盤になるとする。citeturn5search16 成人では、自己報告尺度（例：ECR）や半構造化面接（Adult Attachment Interview）が、内的作業モデル（internal working model）や語りの整合性を通じて、親密関係・対人期待（信頼の形成と親和）を測る。citeturn14search10turn14search11  
- 何としての信頼：**関係的安全（情動調整の資源）＋期待（他者は応答してくれる）**。  
- 測れる/測れない：愛着指標は「特定の他者（養育者・パートナー）との関係史」に強いが、制度やAIなど非人格対象への信頼に直結させるには中間理論が必要。citeturn5search16turn14search11

学校改善研究では、学校共同体における「relational trust（関係的信頼）」が改革の資源として論じられ、教師・保護者・管理職の相互期待として整理される。citeturn7search13turn9search18 教育領域のレビューは、信頼の「性質・意味・測定」を多分野的に整理しようとしている。citeturn9search18

**（可能なら）宗教学**  
宗教学では“信（faith）”が中心になりやすく、哲学的には trust/confidence/faith の境界が論点になりうる。ただし本レビューの一次収集では、宗教学の信頼概念を横断レビューとして統合した高品質ソースを十分に確保できていない（未確認）。  

**（可能なら）進化心理学／進化生物学（協力の進化）**  
進化理論では、協力が維持される条件（互恵性、反復相互作用、裏切り検出など）がモデル化され、信頼を“協力が合理的になる環境条件”として間接的に位置づける。citeturn7search2turn7search15 互恵的利他行動の理論は「裏切り者を排除・罰する仕組み」が協力を安定化させることを論じ、信頼/不信の発達（警戒と委任のバランス）と噛み合う。citeturn7search2turn7search3  
- 何としての信頼：**協力を可能にする条件（生態学的・戦略的）**。  

**（可能なら）情報システム（e-commerce/オンライン信頼）**  
オンライン環境では、相互作用の欠如、匿名性、プラットフォーム構造が信頼を再編し、信頼の多次元測定化が進んだ。e-commerce の信頼尺度整備は「定義の多様性・不一致」を問題化し、統合モデルで測定を提案している。citeturn6search12turn6search0 また、TAM（技術受容モデル）と trust を統合して購買・使用 intention を説明する研究が定番化した。citeturn6search17turn6search5

## 神経現象学と心理学の位置づけ

神経現象学の核心は、「信頼の研究」を（a）行動指標（例：trust game の送金）や（b）生理・神経指標（例：fMRI・心拍）だけに還元せず、**当人が経験する“信頼している瞬間”の構造**（時間経過、注意の向き、身体感覚、意味づけ、他者表象）を、訓練された方法で取り出し、同時計測データの解析にも反映させる点にある。citeturn0search1turn11search0turn3search6 ここで心理学は、(1) 信頼を構成するサブ成分（認知ベース／情緒ベース、予測可能性／頼もしさ／faith 等）の理論と尺度、(2) 発達・臨床（愛着・裏切り・修復）に関する知見、(3) 実験課題設計（相互行為・協力・リスク）を供給する“中核接続点”になる。citeturn1search1turn9search5turn5search16turn3search1

**1人称経験×3人称データの結びつけ方（設計の骨格）**  
神経現象学の実装は、一般に次のような手順で説明される（信頼研究に合わせて具体化する）。

1) **経験対象の明確化（エピソード単位）**  
「信頼して送金を決めた瞬間」「警戒がほどけた瞬間」「裏切りの兆しを感じた瞬間」など、短い時間窓で扱える経験単位を定める。これは信頼を“特性”ではなく“状態”として観察する入口になる。citeturn11search0turn3search22  

2) **経験の高解像度抽出（micro-phenomenology / elicitation）**  
entity["people","Claire Petitmengin","micro-phenomenology method"]らのインタビュー技法は、当人が普段は気づかない経験要素（注意の遷移、感覚モダリティ、暗黙の意味づけ）を、誘導のバイアスを抑えつつ言語化させることを狙う。citeturn3search6turn3search22turn11search21 信頼研究では、たとえば「安心（身体が緩む）」「相手の善意の確信」「監視したい衝動の低下」「自己開示したい衝動」など、感情・身体・意味の束として経験記述を得る。

3) **時間同期（time-locking）と多層データ化**  
経験報告で同定された時間点（例：決断直前の“飛躍”）に合わせて、(a) 行動（反応時間、選択）、(b) 生理（心拍変動、皮膚電気反応）、(c) 神経（EEG/fMRI）、(d) 相互行為ログ（相手の反応、会話）を同期させる。信頼ゲームの神経研究が行う相互行為データ化は、この設計に取り込みやすい。citeturn2search6turn3search0turn3search11  

4) **相互拘束（mutual constraints）**  
経験カテゴリー（1人称）が、3人称解析（例：イベント分割、条件設定）を導き、逆に3人称の規則性・逸脱（例：特定局面での線条体反応の変化）が、経験カテゴリーの再精緻化を促す。これがVarelaの提案する神経現象学の方法論的要点である。citeturn0search1turn0search5  

5) **心理学理論への“逆流”**  
得られた経験構造が、「情緒ベース信頼」「認知ベース信頼」など既存尺度の下位過程（何が“情緒”なのか）を再定義し、質問紙項目や実験操作の改善につながる。組織論でも、信頼（態度）と信頼行動（リスク行為）を区別する枠組みがあるため、経験構造をこの区別へ接続しやすい。citeturn14search5turn1search1  

**特別課題（E：神経現象学での信頼）**  
- **生態学的妥当性**：信頼は多くの場合、関係史・権力・制度に埋め込まれる。実験課題（trust game）を用いる場合、経験報告が「ゲーム文脈の信頼」へ過度に限定されないよう、現実場面のエピソード聴取やエスノグラフィーと併用する必要がある。citeturn2search5turn10search4turn10search8  
- **“感情としての信頼”の同定問題**：安心・好意・期待・同一化・義務感などが混ざるため、経験カテゴリーを「情動ラベル」ではなく「時間構造＋身体＋評価＋行為傾向」で定義する方が、学際的翻訳に強い。citeturn3search22turn1search1turn10search13  
- **生物学的指標の解釈**：オキシトシン研究の追試不一致は、単一生物指標に“信頼”を還元する危険を示す。条件依存・集団境界・社会接触の扱いが不可欠である。citeturn13search1turn13search2

## 主要論争

**信頼は感情か、信念か、期待か、関係か**  
組織論の標準定義（脆弱性を引き受ける意志）や哲学の条件整理は、信頼を「感情」に限定せず、態度・規範・期待を含む複合体として扱う。citeturn14search5turn0search4 一方、情緒ベース信頼などの概念は、信頼の“感情的結合”を理論上区別できることを示すが、それが基本情動（fear等）と同型か、関係情動（安心・親和）として別種かは、分野横断で未統一である。citeturn1search1turn1search13  

**信頼とリスクの関係（信頼≠リスク選好？）**  
Mayerらは、信頼（態度）と信頼行動（リスクを伴う行為）を区別し、リスクは後者に内在すると整理する。citeturn14search5turn1search0 trust game もまた、送金＝“信頼”と解釈されるが、送金は利他性・規範・戦略期待の混合でありうるため、信頼の純度は設計依存である。citeturn2search5turn13search6  

**信頼と不信：単一連続体か、独立次元か**  
信頼/不信を単一軸で測ると「不信が低い＝信頼が高い」と誤読しやすい。組織論では、信頼と不信は概念的に異なり、同一関係内で同時に共存しうるとする枠組みが提示されている。citeturn3search1turn3search21 医療・公衆衛生の mistrust 研究でも、信頼欠如とは別に「搾取される/軽視される」という期待が独立して健康行動に影響するという議論が展開される。citeturn12search8turn12search4  

**信頼と信頼worthiness（信頼に値する性質）の違い**  
AI領域では、trust（人の心理状態）とtrustworthiness（システムの性質・ガバナンス）を分ける必要がとくに強調される。citeturn6search7turn11search10 この峻別は、医療（医師の専門性・誠実性の評価）や政治（政府の誠実性・公正性の評価）にも一般化できるが、分野ごとに評価軸が違う。citeturn4search3turn12search10  

**測定の論争：質問紙 vs 行動課題 vs 生体指標**  
一般的信頼の調査（WVS項目等）は国際比較に強いが、質問が指す“most people”の解釈差や文化差が問題になる。citeturn5search11turn5search15 trust game は行動指標として魅力的だが、反復計測での不安定性、プロトコル依存、外的妥当性の範囲が論点となる。citeturn13search6turn2search5turn13search3 神経・ホルモン指標は機序仮説を与える一方、オキシトシン効果の追試問題が示すように、単純化と過剰一般化のリスクがある。citeturn13search1turn13search20  

**信頼の修復と裏切り**  
裏切りや制度的不正は信頼を破壊し、修復戦略（謝罪、説明責任、補償、手続き改革）を巡って分野横断の研究がある。哲学は説明責任の逆効果（過剰監査）を論じ、組織論は信頼/不信のダイナミクスを扱い、臨床は betrayal trauma の長期影響を扱う。citeturn8search10turn3search1turn9search7  

**人間への信頼とAI/自動化への信頼は同じか**  
自動化信頼は「理解が不完全でも依存する」意思決定であり、対人信頼と類似点をもつが、対象が“意図をもつ他者”か“設計物”かで、脆弱性・裏切りの意味が変わる。citeturn0search2turn0search6turn6search7 ここは概念輸入（対人信頼理論をAIへ）に慎重さが必要な領域である。citeturn0search3turn11search10

1. **「信頼の情動成分」を分解する：安心・好意・期待・義務感・同一化の区別**

初期研究と登録追試の不一致を前提に、社会接触条件、内集団/外集団、個人差の交互作用を系統的に見直す。citeturn1search3turn13search1turn13search2
