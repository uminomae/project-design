# 信頼修復の横断比較：対人・臨床・組織・制度・AI をまたぐ trust repair

Issue: #20  
Parent: #6  
作成日: 2026-03-26  
作成: Codex

## 0. エグゼクティブサマリー

信頼修復は、単に「謝ること」でも「元通りに戻すこと」でもない。学際的に見ると、trust repair は少なくとも四つの要素から成る。第一に、何が壊れたのかを見極めること。能力への期待が壊れたのか、誠実性への期待が壊れたのか、善意への期待が壊れたのか、制度全体の公正性や説明責任への期待が壊れたのかで、必要な応答は変わる。第二に、違反の帰属を処理すること。失敗が偶発的か、過失か、欺瞞か、構造的問題かで repair strategy は変わる。第三に、言語的応答だけでなく、将来の再発可能性を下げる具体策が必要であること。第四に、修復後の trust はしばしば「元の trust」と同じではなく、再編成された trust になることだ。

組織論と対人信頼研究では、この点が最も明確に整理されてきた。Kim らや Schweitzer らの研究は、competence-based violation と integrity-based violation の違い、apology と denial の条件依存性、deception を含む違反の回復困難性を示してきた。Gillespie と Dietz はさらに、組織レベルでは trust repair は dyad の問題ではなく、システム全体の診断と改革を含む multilevel process だと論じた。2021 年の systematic review でも、職場 trust repair 研究の大半が verbal response に偏っており、substantive response や organizational reform の研究が相対的に少ないことが確認されている。

臨床と愛着の文脈では、repair はもっと身体化され、関係的である。裏切りトラウマや慢性的な応答不全のもとで損なわれるのは、単なる「相手を高評価する態度」ではなく、再び vulnerability を開ける capacity そのものである。このため、trust repair は「説明を理解した」だけでは成立せず、「もう一度、相手の前で防御を緩められるか」という閾値の再学習を含む。神経現象学的に言い換えると、repair は belief revision だけでなく、監視トーン、委譲感、身体の硬さ、撤退衝動の変化でもある。

制度・公衆衛生・科学の trust repair はさらに難しい。個人間関係と違い、誰が謝るのか、誰が責任を負うのか、何をもって再発防止とみなすのかが曖昧になりやすいからである。2025 年の公衆衛生 trust review でも、194 本中 trust repair を直接扱った研究はごく少数だった。これは制度 trust の修復が重要でないという意味ではなく、むしろ重要なのに研究上まだ未整備だという意味である。

AI / automation では、repair はしばしば recalibration として現れる。ここで修復されるのは moral trust というより、依存の適切な水準である場合が多い。Kox らの human-agent team 研究は、agent の失敗後、regret と explanation を組み合わせた応答が回復に効く可能性を示した。ただし、AI の場合は apology 自体が本物の remorse を意味しないこと、失敗がシステム設計・データ・運用・組織責任に分散することから、repair は対人的謝罪モデルでは足りない。性能境界の再説明、監督点の再設計、uncertainty communication、accountability chain の可視化が重要になる。

この横断比較から見える結論は明確である。trust repair は共通して、1) violation type の診断、2) 責任の明確化、3) affective acknowledgement、4) substantive change、5) 反復的一貫性、6) 再依存の再学習、を必要とする。ただし各分野で前景化する要素は違う。対人・組織では attribution と apology の条件依存性、臨床では安全感と身体化された再開放、制度では structure reform、AI では recalibration が中心となる。

## 1. 何を修復しているのか

### 1.1 trust violation と reliance failure の違い

- **reliance failure** は、期待した機能が果たされなかった失敗であり、対象の motive を必ずしも問わない。
- **trust violation** は、能力、誠実性、善意、公正性、責任感など、相手または制度の trustworthiness に関する期待が損なわれる失敗である。
- したがって、単なる故障と裏切りは同じではない。後者では betrayal、suspicion、mistrust が前景化しやすい。

### 1.2 何が壊れたかで repair は変わる

- **competence violation**: 失敗したが、悪意や不誠実さは確定していない
- **integrity violation**: 嘘、隠蔽、責任逃れ、不正
- **benevolence violation**: 利己的・冷淡・搾取的なふるまい
- **systemic violation**: 事故や不祥事の背後に組織構造・制度設計の欠陥がある

Kim らの古典研究は、competence と integrity の違いが応答戦略の効果を左右すると示した。

## 2. 対人・組織レベルの trust repair

### 2.1 apology, denial, promise

- Kim, Ferrin, Cooper, Dirks (2004) は、competence-based violation と integrity-based violation で apology と denial の効果が異なることを示した。
- 要点は、どの応答が効くかは violation の種類と、その後に出る guilt / innocence 情報に依存することだ。
- Schweitzer, Hershey, Bradlow (2006) は、deception を含む違反では trust は回復しにくく、謝罪や promise だけでは元に戻りにくいことを示した。

### 2.2 short-term と long-term

- Lewicki & Brinsfield (2017) は trust repair を short-term strategy と long-term strategy に分けて整理した。
- short-term は説明、謝罪、否認、弁明など。
- long-term は、一貫した行動、再発防止、ルール変更、関係再交渉など。
- 横断的に見ても、長期的 repair の方が再依存に効くが、研究は短期反応に偏りがちである。

### 2.3 system-level repair

- Gillespie & Dietz (2009) は、organization-level failure 後の trust repair を、即時対応、原因診断、制度改革、評価の四段階で整理した。
- ここでは apology だけでなく、leadership replacement、policy reform、process redesign、monitoring reform が repair の本体になる。
- 2021 年の systematic review でも、empirical studies の多くは verbal response に偏り、organizational reform の実証は少ない。

## 3. 臨床・愛着レベルの trust repair

### 3.1 裏切りトラウマ

- betrayal trauma では、危害そのものと「頼るべき相手が危害源だった」ことが重なる。
- そのため repair は、単なる認知的納得ではなく、再び vulnerability を開ける capacity の再構築を要する。

### 3.2 repair は安全感の再学習

- 愛着と臨床の観点では、repair は「相手は変わった」という命題理解だけでなく、「いまここで相手の前で身体を硬くしなくてよい」という学習でもある。
- 近接研究からの推定として、repair を支えるのは一貫性、予測可能性、責任承認、反復的な非侵襲経験である。

### 3.3 therapeutic alliance と epistemic trust

- 治療関係では、alliance があっても epistemic trust が回復していない場合、情報は深く取り込まれない可能性がある。
- したがって臨床 repair は、関係を保つことと、他者から受け取ることを再び可能にすることの二層構造を持つ。

## 4. 制度・公衆衛生・科学の trust repair

### 4.1 制度 repair の難しさ

- 誰が trust violator かが曖昧
- 単一行為者ではなく複数組織・複数レベルが関与
- 何をもって誠実性とみなすかが contested

### 4.2 公衆衛生の現状

- 2025 年の scoping review では、194 本中 trust repair に直接焦点を当てた報告はごく少数だった。
- これは、制度信頼の修復が未研究であることを示す重要な発見である。
- とくに公衆衛生では、repair は message correction だけでは足りず、historical injustice、公平性、手続き的正義、情報環境の改善を要する。

### 4.3 制度 repair の要件

- 責任の明確化
- 透明な調査
- reform の visible implementation
- 公平性と respect の回復
- performance の継続的一貫性

## 5. AI / automation における repair

### 5.1 repair は recalibration に近い

- AI での trust repair は、対人的 remorse の回復というより、「どこまで依存してよいか」を再調整する process として理解しやすい。
- そのため explanation, regret, transparency, post-failure performance が中心になる。

### 5.2 human-agent teams の知見

- Kox ら (2021) は、agent の trust violation 後、regret を含む応答で trust が回復しやすく、explanation が加わると効果が強まる可能性を示した。
- ただし advice-taking まで一貫して回復するとは限らず、態度的 trust と行動的 reliance は分けて見る必要がある。

### 5.3 AI repair で不足しやすいもの

- 真の責任主体が不明なままの apology
- explanation が reassurance に偏り calibration を歪めること
- 単発の UI response に依存し、system trustworthiness の改革が置き去りになること

## 6. 神経現象学から見た trust repair

### 6.1 repair の経験構造

- 警戒がどれくらい下がるか
- 身体の硬さがどれくらい戻るか
- 相手を再び見られるか
- 委譲感が復活するか
- 「またやられるかもしれない」という予期がどう変わるか

### 6.2 近接研究からの推定

- trust repair は belief revision だけではなく、防御トーンの再調整である可能性が高い
- よって、repair を測るには self-report だけでは弱く、経験記述と行動・生理の組み合わせが望ましい

## 7. 測定法と限界

- **質問紙**: repair 後の態度変化は見やすいが、再依存の閾値は見えにくい
- **実験ゲーム**: 行動変化は見えるが、betrayal と mere failure の主観差は粗い
- **縦断職場研究**: reform の効果を見やすいが、介入の因果推定が難しい
- **臨床面接**: repair の質感を捉えやすいが、標準化しにくい
- **micro-phenomenology**: 再び委ねられる感じを捉えうるが、研究蓄積はまだ薄い
- **AI log analysis**: recalibration を見やすいが、なぜ再依存したかは分からない

## 8. 横断比較の暫定結論

### 8.1 共通する条件

1. violation type の診断  
2. 責任の明確化  
3. affective acknowledgement  
4. substantive change  
5. 反復的一貫性  
6. 再依存の再学習  

### 8.2 分野別の違い

- **対人 / 組織**: attribution と apology / denial の条件依存性
- **臨床 / 愛着**: vulnerability と安全感の再構築
- **制度 / 公衆衛生**: reform と公正性の可視化
- **AI / automation**: recalibration と trustworthiness communication

### 8.3 重要な留保

- repaired trust は pristine trust と同じでない場合が多い
- deception を含む違反は repair が特に難しい
- institutional trust repair と neurophenomenological trust repair は、研究上まだ未整備である
