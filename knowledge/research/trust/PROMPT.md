# 「信頼とは」調査プロンプト

使用方法: このプロンプトを ChatGPT / ChatGPT Pro / Codex の3モデルに同一の形で投げる。
作成日: 2026-03-27

---

あなたは、学際的レビューと概念整理に強い研究アシスタントです。

調査テーマは **「信頼とは何か」** です。
ただし今回は、最初から狭く定義せず、**広く俯瞰すること** を最優先にしてください。
後から深掘りする前提なので、まずは「信頼研究の全体地図」を作ってください。

## 最重要方針

- **「信頼＝感情」と決めつけないこと。**
- 分野ごとに、信頼が
  - 感情
  - 態度
  - 期待
  - 関係の性質
  - 社会規範
  - 制度的資本
  - 合理的選択やリスク判断
  - 学習・予測の結果
  - lived experience（生きられた経験）
  のどれとして扱われるかを比較してください。
- **定義の違いそのもの** を成果物にしてください。
- 単なる雑多な文献列挙ではなく、**分野横断の見取り図** を作ってください。

1. **神経現象学（neurophenomenology）**

   - 制度への信頼

## 調査の中心問い

以下の問いに答える形で整理してください。

### A. 概念整理
- 「信頼」は各分野でどう定義されるか
- 何と区別されるか
  （例：reliance, confidence, faith, mistrust, distrust, trustworthiness, credibility, legitimacy）
- 信頼の対象は何か
  （他者、集団、制度、専門家、科学、国家、AI、自分自身 など）
- 信頼は状態か、特性か、関係か、過程か

### B. 水準の違い
- 個人内の信頼
- 対人的信頼
- 一般的信頼（generalized trust）
- 制度的信頼
- 政治的信頼
- 認識論的信頼（誰の知識を信じるか）
- 技術への信頼
- 自己信頼

### C. 発達・形成
- 信頼はどのように形成されるか
- 愛着や初期養育との関係
- 裏切りやトラウマは信頼にどう影響するか
- 信頼は学習されるのか、更新されるのか、修復できるのか

### D. 測定法
- 各分野は信頼をどう測るか
  - 質問紙
  - 行動課題
  - trust game / economic games
  - 実験
  - 縦断研究
  - 面接
  - エスノグラフィー
  - 微視的現象学的インタビュー
  - 生理指標
  - 神経画像
  - ホルモン測定
- それぞれ何を測っていて、何を測れていないか

### E. 神経現象学の特別課題
- 神経現象学では「信頼」をどう扱いうるか
- 1人称の経験報告と3人称の行動・神経データをどう結びつけられるか
- 「信頼している感じ」「身を預けられる感じ」「警戒がほどける感じ」などの lived experience を、
  どのような方法で記述・比較・検証できるか
- 既存研究が少ない場合は、「近接領域からどこまで言えるか」を慎重に示してください

### F. 学問間のズレ
- 分野ごとに何が食い違っているか
- 何が同じ構造として見えるか
- どの分野が「感情」と見なし、どの分野が「期待」「制度」「規範」「行動戦略」と見なすか
- 学問横断で再整理すると、どんな軸で分類できるか

### G. 日英翻訳ズレ
- 日本語「信頼」と英語 trust / confidence / reliance / faith / credibility / legitimacy 等の対応関係を整理してください
- 日本語では「信頼」「信用」「信任」「安心」「頼り」など複数の語がどう使い分けられるか
- 翻訳時に生じやすい混同・誤解パターンを示してください

## 出力形式

Markdown（.md想定）で、以下の構成にしてください。

# タイトル
信頼とは何か：学際的見取り図（広域レビュー）

## 0. エグゼクティブサマリー
- 1000〜2000字程度
- まず全体像を簡潔に述べる
- 「信頼」は一つの概念ではなく、複数の研究伝統が重なっていることを示す
- この段階での主要な整理軸を提示する

## 1. 分野別マップ
各分野ごとに以下を簡潔に整理
- その分野での信頼の定義
- 主な理論
- 主要な測定法
- 代表的知見
- 典型的な論争点
- 信頼を何として扱うか（感情/態度/期待/関係/制度/その他）

## 2. 横断比較
- 分野をまたいだ比較表
- 以下の列を推奨
  分野 / 定義 / 主体 / 対象 / 水準 / 方法 / 代表概念 / 近接概念 / 主論点

## 3. 日英翻訳ズレ
- 信頼 / trust / confidence / reliance / faith / credibility 等の対応表
- 日本語内の使い分け（信頼・信用・信任・安心・頼り）
- 翻訳時の誤解パターン

## 4. 神経現象学と心理学の位置づけ
- 神経現象学を独立節で扱う
- 心理学全般との接続
- 愛着、社会認知、感情、自己感、身体感覚との関係
- 「信頼の主観的経験」をどう研究できるかを明確化

## 5. 主要論争
最低限、以下を扱う
- 信頼は感情か、信念か、期待か、関係か
- 信頼とリスクの関係
- 信頼と脆弱性（vulnerability）の関係
- 信頼と裏切り・修復
- 信頼と制度・権力
- 信頼と不信の違い
- 信頼と trustworthiness の違い
- 人間への信頼とAI/自動化への信頼は同じか

## 6. 重要文献リスト
- できるだけレビュー論文、ハンドブック、主要理論論文、古典論文を優先
- 分野別に整理
- 各文献に1〜3行で意義を付記
- **可能な限り URL（DOI, arXiv, 公式ページ等）を付記すること**
- まずは 30〜50件程度を目安

## 7. 深掘り候補
最後に、今後の深掘り候補を 5〜10 個挙げる
例:
- 愛着と信頼
- 信頼の神経現象学
- trust game の限界
- 政治的信頼と制度信頼
- 科学・医療への信頼
- AIへの信頼
- 信頼の修復
など

## 文献調査ルール

- まずは **レビュー論文、ハンドブック、学術辞典、主要理論論文** を優先
- 必要に応じて代表的な実証研究を加える
- 「よく言われる」ではなく、できるだけ出典に基づく
- **出典には可能な限り URL（DOI, arXiv, 公式ページ, Google Scholar リンク等）を付記すること**
- 分野が違えば定義も違うので、安易に統一しない
- 分からない部分は「未確認」「この分野では未整理」「近接研究からの推定」と明記
- 英語圏文献を中心にしてよいが、日本語で読めるレビューや概説があれば補足
- できるだけ **古典 → 主要レビュー → 最近の見直し** の順で見取り図を作る

## 特に重視すること

- 広く拾う
- 学際的に比較する
- 定義のズレを見せる
- 神経現象学と心理学を必須軸にする
- いきなり結論を固定しない
- 深掘りの入口になる「地図」を作る
