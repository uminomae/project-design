# 愛着と信頼：発達・臨床・認識論的信頼の接続

Issue: #8  
Parent: #6  
作成日: 2026-03-26  
作成: Codex

## 0. エグゼクティブサマリー

愛着と信頼は重なるが同一ではない。愛着理論が主に扱うのは、応答的な他者がいるという予測可能性と安全基地の形成であり、信頼研究が扱うのは、そこから派生する対人的開放、依存の意志、裏切られうる状況での委譲、認識論的受容、自己への依拠までを含む、より広い現象群である。発達研究では Erikson の basic trust と Bowlby/Ainsworth の愛着安全性が出発点となるが、成人の対人信頼、一般的信頼、制度信頼、epistemic trust、self-trust はその単純な延長ではない。

もっとも、愛着は信頼形成の重要な下地である。養育者が一貫して利用可能で、苦痛に対して応答的であるとき、子どもは「世界は完全には安全でなくても、必要時には支えが戻ってくる」という予測スタイルを学びやすい。この予測スタイルは、親密圏での vulnerability 受容、情報源の選別、助けを求めること、関係の修復可能性の見立てにまで影響しうる。他方、不安定・回避的・無秩序な関係史は、対人信頼の低下だけでなく、「近づきたいが委ねられない」「信じたいが監視をやめられない」といった両価的な信頼様式を生みうる。

臨床ではこの問題がより鋭く現れる。裏切りを含むトラウマは、相手の善意期待だけでなく、自己の判断への trust、証言や助言を受け入れる capacity、治療関係における開放性を損なうことがある。この文脈で近年大きな位置を占めるのが epistemic trust である。Fonagy/Luyten 系の議論では、治療変化の一部は「他者から受け取る情報が自分に関連しうる」という開放性の回復によって起こるとされる。ただし、ここでいう epistemic trust は発達心理学の selective trust や社会認識論の testimony trust と部分的に重なりながらも、臨床的にはより関係的・安全感依存的に使われている。

この深掘りから見える要点は四つある。第一に、愛着は trust の前提条件の一部だが十分条件ではない。第二に、trust の破綻は単なる low trust ではなく、裏切り、羞恥、解離、監視亢進、自己不信と絡む。第三に、trust repair は認知的説得だけでなく、身体的・関係的に「再び委ねられるか」の閾値の問題である。第四に、発達・臨床・認識論を横断すると、信頼は感情でもあり期待でもあり、関係モデルでもあり、学習された受容態勢でもある。

## 1. 概念の切り分け

### 1.1 basic trust

- Erikson の basic trust は乳児期の発達課題であり、「世界と他者はおおむね生き延びうる場所である」という原初的安全感に近い。
- これは対人的 trust の前史として重要だが、成人の interpersonal trust や generalized trust とそのまま一致しない。

### 1.2 attachment security

- 愛着安全性は、特定の養育者または親密他者の可用性・応答性への期待である。
- trust のうちとくに「必要時に支えが戻る」「脆弱さを見せても見捨てられない」という側面を強く支える。

### 1.3 interpersonal trust

- 成人の対人信頼は、相手の誠実性・善意・予測可能性・dependability への期待であり、愛着だけでなく関係履歴、評判、制度文脈、自己信頼にも左右される。
- 近接領域からの推定として、愛着は trust propensity と親密圏での trust dynamics に強く関わるが、相手固有の trustworthiness 評価とは切り分ける必要がある。

### 1.4 epistemic trust

- 子ども研究では「誰の情報を信じるか」という selective trust の問題として現れる。
- 臨床では「他者から受け取る情報が自分にとって relevant で、取り込んでよいと感じられるか」という受容性として使われる。
- 哲学では testimony や他者知識への依拠の正当化として扱われる。

### 1.5 self-trust

- self-trust は自己効力感より広く、自分の判断・動機・感覚・記憶・価値判断に対する依拠可能感を含む。
- 愛着不安やトラウマは他者信頼だけでなく self-trust も損なうことがある。

## 2. 形成：初期養育から対人期待へ

### 2.1 応答性と予測可能性

- Bowlby の internal working models は、自己と他者についての反復的期待の枠組みとして理解できる。
- 応答的養育は「苦痛時に相手が利用可能である」という予測を安定化しやすい。
- この予測は、後の help-seeking、自己開示、依存の許容、裏切り後の回復可能性の見積もりに波及しうる。

### 2.2 selective trust との接続

- 発達研究は、子どもが早期から情報源の正確さ、専門性、一貫性を手がかりに selective trust を行うことを示している。
- これは attachment security とは別系統の認識論的信頼だが、近接研究からは、社会的安全感が高いほど情報更新や他者受容が行いやすい可能性がある。

### 2.3 一般化信頼との距離

- generalized trust は「見知らぬ他者一般」に対する期待であり、愛着だけでは説明しきれない。
- 社会的不平等、制度の公正さ、文化規範、共同体経験が大きく関わるため、愛着から一般化信頼への直線モデルには注意が必要である。

## 3. 親密圏での信頼

### 3.1 Rempel-Holmes-Zanna 系の信頼モデル

- close relationships の trust は、predictability、dependability、faith という次元で理解されてきた。
- 2025 年の再検討でも、この三層構造は大筋で維持されつつ、関係初期と長期で細部に差があることが示された。
- ここでの faith は単なる irrationality ではなく、完全証拠がない段階で相手に向けられる関係的賭けに近い。

### 3.2 愛着不安と愛着回避

- 愛着不安は、見捨てられ不安と reassurance 追求を通じて trust を不安定化しやすい。
- 愛着回避は、依存や vulnerability を嫌うため、表面上は安定していても trust の受容局面が狭くなりやすい。
- したがって low trust は一枚岩ではなく、「近づきたいのに委ねられない」と「そもそも委ねない」が分かれる。

## 4. 裏切り、トラウマ、修復

### 4.1 betrayal trauma

- 裏切りトラウマは、危害そのものに加え、頼るべき対象が危害源であったという点で trust 破綻を深める。
- その結果、対人信頼だけでなく、自己の知覚や判断、助けを求める能力、制度への trust まで損なわれうる。

### 4.2 修復の条件

- 修復は謝罪だけでは足りない。責任承認、予測可能な再発防止、時間、一貫した行動が必要である。
- 臨床的には、修復は「もう一度相手に vulnerability を開けるか」という関係的・身体的閾値の問題でもある。
- 近接研究からの推定として、trust repair は認知的再評価と身体的安全感の再学習の双方を含む。

## 5. Epistemic Trust と心理療法

### 5.1 臨床での位置づけ

- Fonagy/Luyten 系では、epistemic trust は他者からの情報を「自分に関連するもの」として受け取り、一般化できる能力として扱われる。
- その障害は epistemic mistrust や epistemic credulity と対置される。

### 5.2 治療同盟との違い

- 治療同盟は協働関係の質を指すが、epistemic trust はその関係を通じて何かを受け取り変化できる状態を指す。
- 強い alliance があっても ET が回復していない場合、情報は受け取られても transform されない可能性がある。

### 5.3 注意点

- 臨床 ET はまだ概念拡張の途上であり、哲学的 testimony trust や子どもの selective trust と完全には整合していない。
- この領域では理論の広がりに対して測定の標準化が追いついていない。

## 6. 測定法

### 6.1 愛着

- Strange Situation
- Adult Attachment Interview
- self-report attachment scales

### 6.2 対人信頼

- Rempel らの close relationship trust scale
- propensity to trust / other-focused trust 尺度
- 関係履歴を伴う縦断 survey

### 6.3 臨床・認識論的 trust

- Epistemic Trust, Mistrust, and Credulity Questionnaire
- 治療同盟尺度
- トラウマ関連面接・症状尺度

### 6.4 測れていないもの

- 身体感覚としての「委ねられる感じ」
- 裏切り後にどの程度監視が残るか
- self-trust と other-trust の相互変換
- 関係修復の閾値の時間的推移

## 7. 主要論争

1. 愛着は trust の原因か、相関する背景条件か。  
2. basic trust と adult trust を同一線上に置いてよいか。  
3. epistemic trust は独自概念か、attachment security の派生か。  
4. low trust は病理か、防衛として合理的か。  
5. 修復は認知レベルで起こるのか、身体化された安全感の再構築を要するのか。

## 9. 未解決点

- 愛着安全性と generalized trust の媒介経路はまだ粗い。
- ET は概念的には強いが、測定上の収束妥当性はなお論争的である。
- self-trust の心理学的測定は sparse で、哲学的議論との接続が弱い。
- 信頼修復を「再び身を預けられる感覚」として追う研究は不足している。
