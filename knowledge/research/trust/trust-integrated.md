## 要約

「信頼」は単一概念ではなく複合概念である。分野ごとに感情、態度、期待、関係の性質、制度的資本、行動戦略、認識論的条件、経験構造として扱われる。共通核は「脆弱性」「期待」「依存」「予測不能性」「監視の限界」「修復可能性」にある。日英間では trust / confidence / reliance / faith の区別が日本語の「信頼」「信用」「安心」に一対一対応しないズレがある。信頼を定義するには「誰が、何を、どの水準で、何として信頼と呼んでいるか」を先に確定する必要がある。

## 「信頼」の多層定義

| 層 | 信頼を何として扱うか | 代表分野 | 英語対応 |
|---|---|---|---|
| 感情 | 安心・好意・身体的弛緩を伴う情動的結合 | 臨床心理学、愛着理論 | affective trust |
| 期待 | 相手が善意・能力をもって行動するという見込み | 社会心理学、組織論 | expectation, belief |
| 関係の性質 | 相互行為に埋め込まれた安定性・安全感 | 社会学、人類学 | relational trust |
| 制度的資本 | 社会秩序・複雑性縮減を支える装置 | 社会学、政治学 | institutional trust, social capital |
| 行動戦略 | 不確実性下で相手に資源を委ねる選択 | 経済学、神経経済学 | trust behavior, investment |
| 認識論的条件 | 他者の証言・情報を受容可能にする前提 | 哲学、臨床(epistemic trust) | epistemic trust |

## 日英翻訳ズレ

| 英語 | 主な日本語対応 | ズレのポイント |
|---|---|---|
| trust | 信頼 | 最も広義。態度・期待・関係を含む |
| confidence | 信頼、確信 | Luhmann: confidence = 代替なき依存（制度的）、trust = リスクを意識した委託 |
| reliance | 依拠、依存 | 哲学: trust は reliance + 裏切り可能性への認識 |
| faith | 信仰、信念 | 証拠を超えた確信。宗教的文脈で前景化 |
| credibility | 信憑性 | 情報源の質の評価。trust より狭い |
| trustworthiness | 信頼に値すること | trustee 側の性質。trust(trustor側)と混同されやすい |
| distrust | 不信 | trust の単なる欠如ではなく、積極的な否定的期待 |
| mistrust | 不信、猜疑 | distrust より拡散的。人類学では正当な社会的構えとして扱う |
| 信用 (JP) | credit, reliability | 実績・能力に基づく評価。山岸: 信用 = 情報に基づく判断 |
| 安心 (JP) | assurance, security | 山岸: 安心 = 制度的保証による不確実性の除去。trust とは別概念 |
| 信認・信任 (JP) | confidence, mandate | 政治・法的文脈での委託関係 |

山岸(1998)の核心的区別: **安心(assurance)** は制度や監視により裏切りが不可能な状態。**信頼(trust)** は裏切りの可能性がある中で相手の善意を期待すること。両者は質的に異なる。

Luhmann(1979)の区別: **confidence** は代替選択肢を意識しない日常的依存。**trust** はリスクを認識した上での能動的委託。

## 近接概念の区別

| 概念ペア | 区別の要点 |
|---|---|
| trust vs confidence | Luhmann: confidence は代替なき依存、trust はリスクを意識した能動的委託 |
| trust vs reliance | 哲学: reliance は機能的依存。trust は相手の善意への期待と裏切り可能性を含む |
| trust vs faith | faith は証拠や実績を超えた確信。trust は証拠・経験に基づく場合も多い |
| trust vs trustworthiness | trust は trustor 側の態度。trustworthiness は trustee 側の能力・善意・誠実性 |
| trust vs distrust | distrust は trust の単なる欠如ではなく、積極的な否定的期待。同一関係内で併存しうる |
| distrust vs mistrust | distrust は特定対象への否定的期待。mistrust はより拡散的な猜疑・警戒のスタンス |
| trust vs credibility | credibility は情報源の質への評価。trust はより広く態度・関係・脆弱性を含む |

## 主要論争

1. **感情か、信念か、期待か、関係か**: 信頼の存在論的カテゴリーは分野により異なる。一つに還元するより「どの水準の現象を問うか」を先に確定すべきとされる (SEP; Rousseau et al. 1998)。

2. **信頼とリスク**: 多くの分野で不確実性・vulnerability を前提とするが、経済学では明示的選択問題、哲学では裏切り可能性と規範性、臨床では身体的・情緒的開放を意味する。trust game への主要批判もここにある (Bohnet & Zeckhauser 2004)。

3. **信頼と脆弱性(vulnerability)**: Baier(1986)以降、信頼を vulnerability から切り離せないとする議論は強い。ただし HCI での vulnerability は「完全監視ができないので依存する」であり、対人的 moral vulnerability とは異なる。

4. **信頼と裏切り・修復**: 裏切りは trust 特有の損傷であり、mere reliance の失敗とは質的に異なる。修復には責任承認・再発防止・時間・反復的一貫性が必要 (Mayer et al. 2007; Platt & Freyd 2015)。

5. **信頼と制度・権力**: 制度信頼は「個人がもっと信じればよい」問題ではなく、制度側の trustworthiness と accountability の問題でもある。低信頼は歴史的不正義への合理的応答でありうる (Levi & Stoker 2000)。

6. **trust と distrust の関係**: ほぼすべての分野で distrust は trust のゼロ点ではない。哲学では積極的否定的期待、政治学では cynicism と healthy skepticism を区別。low trust = 病理とは限らない (Lewicki et al. 1998)。

7. **trust と trustworthiness の非対称性**: trust(trustor 側)と trustworthiness(trustee 側)は別概念。多くの測定上の混乱はこの混同から生じる。AI 研究・医療・組織論で特に重要 (Mayer et al. 1995; NIST AI RMF)。

8. **人間への信頼と AI/自動化への信頼**: AI には善意や道徳的責任主体性を帰属しない立場も強い。完全同一でも完全別物でもなく、部分重複モデルが妥当。擬人化により対人信頼メカニズムが部分的に流入する (Lee & See 2004; Glikson & Woolley 2020)。

## 探索すべき問い

- **信頼は状態か、特性か、関係か、過程か**: 分野により前提が異なり、未統合。同一個人内でも状態と特性は分離すべきか。
- **信頼の形成と修復は同じメカニズムか**: 修復は単なる再形成ではなく、「もう一度警戒を解けるか」という固有の閾値問題を含む可能性がある。
- **epistemic trust は対人信頼と同じ構造か**: 臨床の epistemic trust と哲学の testimony 論は重なるが同一ではない。発達起源との接続も未整理。
- **文化による信頼概念の差異**: 高信頼社会 vs 低信頼社会の二分法は粗い。山岸の安心/信頼区別は文化差の再解釈を可能にする。
- **神経現象学で「信頼の lived experience」をどう研究できるか**: 方法論的提案は強いが実証蓄積はまだ疎。「警戒がほどける感じ」の経験構造記述と神経・生理データの対応が課題。
