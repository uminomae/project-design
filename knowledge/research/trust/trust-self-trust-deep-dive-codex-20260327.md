# 自己信頼：愛着・臨床・認識論・フェミニスト哲学を横断する独立レビュー

Issue: #21  
Parent: #6  
作成日: 2026-03-27

## 0. エグゼクティブサマリー

自己信頼は、単なる自信でも、自己効力感でも、自己肯定感でもない。学際的に見ると、自己信頼とは「自分の判断、感覚、記憶、価値づけ、推論、欲求の読み取りに、どの程度依拠してよいか」に関わる reflexive trust の問題である。trust の対象が他者ではなく自分自身であるため、trustor と trustee が分離しないという特異性を持つが、だからといって単純化されるわけではない。むしろ、自己信頼には、内的経験の interpretive authority、自分の認識能力への fallible な依拠、将来の自分との協働、自律性の条件、社会的抑圧や gaslighting による侵食、トラウマ後の修復可能性といった多層の論点が重なる。

哲学では少なくとも三つの主要伝統がある。第一に Lehrer 系の自己評価ループとしての self-trust である。ここでは自己信頼は、知識や合理的判断の基盤として不可避の reflexive commitment に近い。第二に、Govier や McLeod 以降のフェミニスト哲学・relational autonomy の流れである。ここでは自己信頼は、自律の条件であると同時に、抑圧、ケア関係、医療権力、再生産医療、社会的軽視によって損なわれうる社会的に形成された能力として扱われる。第三に、現代社会認識論・epistemology の流れで、Zagzebski、Dormandy、Michaelian らが、epistemic self-trust を testimony や disagreement、higher-order evidence と同列の問題として論じている。

心理学と臨床では、自己信頼という語そのものは哲学ほど中心ではない。しかし近接領域は厚い。自己効力感、self-concept clarity、cognitive insight、metacognition、attachment security、epistemic trust、trauma and betrayal、gaslighting はいずれも、自己信頼の部分構成または損傷経路として読める。とくに trauma や gaslighting は、自己信頼の破壊を「自分の知覚・記憶・判断を、自分自身で信用できなくなること」として浮かび上がらせる点で重要である。ここでは損なわれるのは self-esteem のみではなく、世界を読むための内的足場そのものだ。

このレビューから見える大きな点は五つある。第一に、自己信頼は global positive self-regard ではない。自分を好きでなくても、自分の判断に一定の trust を持つことはありうるし、その逆もある。第二に、自己信頼は domain-specific でありうる。道徳判断には自信があるが感情理解にはない、記憶には弱いが価値判断には強い、といった不均一性がある。第三に、自己信頼は社会的に形成される。ケア、尊重、抑圧、証言の扱われ方、gaslighting、専門家との関係はすべて自己信頼に影響する。第四に、自己信頼の損傷は臨床的にはトラウマ、解離、認識論的閉鎖、他者依存の過剰化や逆に極端な自己防衛とつながりうる。第五に、修復は introspection だけでは起こりにくく、信頼できる他者や制度に媒介された relational repair を要することが多い。

## 1. 概念の整理

### 1.1 自己信頼とは何か

- 自分の認知能力、価値判断、感情理解、記憶、欲求解釈に対して、どの程度依拠してよいかに関する reflexive trust。
- fallibility を前提としつつ、それでもなお自分の側から判断を進めざるをえないという実践的構え。
- 他者への trust と同様に vulnerability を含むが、その vulnerability は「自分が誤るかもしれないのに、自分を出発点にせざるをえない」という形をとる。

### 1.2 何と区別されるか

- **self-confidence**: 主観的な自信や certainty。自己信頼より狭く、誤った確信でも高くなりうる。
- **self-efficacy**: ある課題を遂行できるという期待。Bandura 的には行動能力の予期であり、自己信頼の一部だが全体ではない。
- **self-esteem**: 自分への価値評価。自分を価値ある存在と思うことと、自分の判断を信用できることは別である。
- **self-respect**: 自分を規範的主体として扱うこと。自己信頼と近いが、評価軸がより規範的。
- **authenticity**: 自分の価値や欲求に忠実であること。自己信頼がなければ authenticity は支えにくいが、同一ではない。
- **cognitive insight**: 自分の思考の誤り可能性を認識し修正する能力。臨床的自己信頼の近接概念。

## 2. 主要伝統

### 2.1 Lehrer: ループとしての自己信頼

- Lehrer の自己信頼論では、知識や合理的受容は最終的に自己評価ループに支えられる。
- 疑ってもなお、どこかで自分の評価能力に trust しなければ判断は始まらない。
- この系譜では自己信頼は epistemic starting point に近い。

### 2.2 Govier / McLeod: 自律と抑圧

- Govier は self-trust を autonomy や self-esteem と接続しつつ、自己信頼の損傷が他者・制度・社会現実の構築に関わることを論じた。
- McLeod は reproductive autonomy 文脈で、女性の self-trust が医療権力、技術、ジェンダー化された関係性によって undermined されることを示した。
- ここでは self-trust は私的能力ではなく relationally shaped capacity である。

### 2.3 Zagzebski / Dormandy / Michaelian: 現代認識論

- Zagzebski は epistemic self-trust を epistemic authority の条件として扱う。
- Dormandy は self-trust を mere reliance ではなく person-directed な normatively rich trust に近づけて捉える。
- Michaelian は metacognition と memory の文脈で、社会的 trust と self-trust の並行性を論じる。

## 3. 愛着・発達との接続

### 3.1 愛着安全性と自己信頼

- 愛着研究は通常 self-trust を中心語にしないが、secure attachment は「自分の感情やニーズは応答に値する」という感覚を支えやすい。
- この点で、愛着安全性は self-trust の発達的土台の一部と読める。
- ただし generalized self-trust まで直線的に導くとは限らず、後続の関係経験や抑圧経験が大きく介入する。

### 3.2 selective trust との違い

- 子どもの selective trust は「誰の情報を信じるか」の問題であり、自己信頼ではない。
- ただし、自分で情報源を選別できる感覚は、認識論的 self-trust の萌芽として読める可能性がある。
- これは近接研究からの推定であり、未整理部分が大きい。

## 4. 臨床・トラウマ・gaslighting

### 4.1 トラウマ

- betrayal trauma や chronic invalidation は、他者信頼だけでなく self-trust を損なう。
- 典型的には「自分の感覚を信用してよいか分からない」「自分の記憶が正しいと思えない」「自分の怒りや不快の正当性が読めない」といった形で現れる。
- これは self-esteem 低下だけでは説明しにくい。

### 4.2 gaslighting

- gaslighting 研究は、自己信頼を損なう典型的プロセスを示す。
- Spear は gaslighting を、peer disagreement と epistemic injustice の文脈で self-trust を掘り崩す実践として整理した。
- ここでは damage は「自分の判断が誤っているかもしれない」ではなく、「自分の判断を出発点にしてよいという feeling of entitlement」が失われることに近い。

### 4.3 臨床での近接概念

- cognitive insight
- metacognitive capacity
- self-concept clarity
- epistemic mistrust / credulity
- self-doubt and experiential invalidation

これらは自己信頼の代理指標にはなりうるが、完全な代替ではない。

## 5. 自己信頼の修復

### 5.1 修復は個人内だけで起こらない

- self-trust は自分の内面だけで完結しないため、修復も他者や制度に媒介されることが多い。
- とくに trauma や gaslighting の後では、外部からの consistent validation と non-invasive response が足場になる。

### 5.2 修復の候補条件

1. 自分の経験が否認されないこと  
2. 誤り可能性を認めつつ、全面的自己否認に落ちないこと  
3. domain-specific に小さく再依拠できる場面を作ること  
4. 失敗しても自己信頼全体が崩壊しないフレームを持つこと  
5. 他者信頼と自己信頼を二者択一にしないこと  

### 5.3 修復と自律

- フェミニスト哲学では、自己信頼の回復は autonomy recovery と重なる。
- ただし autonomy を「一人で完結する能力」と見ると、かえって自己信頼の relational dimension を見失う。

## 6. 測定法とその限界

### 6.1 直接測定

- 2026-03-27 時点で、自己信頼そのものを標準化して測る心理尺度は未整理または sparse。
- 哲学文献では概念分析が中心で、心理測定は限定的。

### 6.2 近接尺度

- self-efficacy scales
- self-esteem scales
- self-concept clarity
- Beck Cognitive Insight Scale
- Epistemic Trust, Mistrust, and Credulity Questionnaire

### 6.3 測れていないもの

- 自分の知覚や価値判断に対する「依拠してよい感じ」
- gaslighting 後の interpretive authority の回復
- self-trust と other-trust の相互補強 / 相互破壊
- domain-specific self-trust の非対称性

## 7. 主要論争

1. 自己信頼は self-confidence の洗練版か、それとも別概念か。  
2. self-trust は epistemic concept か、moral-psychological concept か。  
3. self-trust は autonomy の条件か、結果か。  
4. self-trust は他者信頼に先立つのか、他者信頼の中で形成されるのか。  
5. gaslighting や抑圧で損なわれる自己信頼は、個人療法だけで修復できるのか。

## 9. 暫定結論

- self-trust は self-esteem や self-efficacy に還元できない独立概念である。
- 哲学では理論蓄積が厚いが、心理学・臨床では近接概念に分散しており、測定は未整理である。
- self-trust の形成と損傷は本質的に relational であり、愛着、抑圧、医療、gaslighting、トラウマが深く関わる。
- self-trust の修復は、他者への再信頼と並行して進む場合が多く、自己内省だけでは足りない。

## 10. 次の論点

1. **trust の lived experience taxonomy**  
   self-trust を含め、信頼経験の項目化へ進む。

2. **gaslighting と trust の横断整理**  
   other-trust, self-trust, institutional trust の三層でまとめる。

3. **自己信頼の測定設計**  
   近接尺度を寄せ集めるのでなく、self-trust 固有の experience-based measure を設計する。
