# 信頼とは何か：学際的見取り図（広域レビュー）

調査日: 2026-03-26  
作成: Codex

本稿は、信頼研究を最初から一つの定義に閉じず、主要分野ごとの定義・対象・測定・論争を並べて比較するための広域レビューである。レビュー論文、ハンドブック、古典理論、代表的な方法論文を優先し、必要に応じて実証研究を補った。特に神経現象学については、信頼そのものを正面から扱う既存研究がまだ薄いため、近接領域からの慎重な推定を明示する。

## 0. エグゼクティブサマリー

「信頼」は単一概念ではない。少なくとも、哲学では「脆弱性を引き受けて相手に身を委ねること」として、心理学では「他者の善意・信頼性への期待」や「安定した信頼傾向」として、組織論では「リスク下で相手に依存する意志」として、経済学では「不完備契約下でのリスクある委譲行動」として、社会学では「複雑性を縮減する社会的メカニズム」や「一般化された期待」として、政治学では「市民が制度や政治主体に特定の行為を期待する関係」として、HCI では「自動化や AI にどこまで依存するかを媒介する心理状態」として、臨床・発達では「養育・愛着・裏切り経験により形成される安全感や対人的開放性」として扱われてきた。神経科学ではさらに、信頼は単独の心的実体というより、予測、報酬、社会評価、脅威低減、学習、相互行為調整の複合過程として分解される。

したがって、「信頼＝感情」と固定するのは不十分である。感情は確かに重要な層だが、それだけではない。ある分野では信頼は態度であり、別の分野では期待であり、別の分野では関係の質であり、制度論では社会資本や正統性を支える媒介であり、経済学では不確実性下の選択と相互期待の表出であり、発達・臨床では安全の学習とその破綻・修復の問題であり、神経現象学的には「警戒がほどける」「身を預けられる」「相手の不透明さに耐えられる」といった lived experience の組織化として捉えうる。

この広域レビューから見えてくる主要整理軸は少なくとも六つある。第一に、信頼を何として扱うかという軸である。感情、態度、期待、関係、社会規範、制度的資本、合理的賭け、学習された予測、認識論的開放性、身体化された経験は互いに置換可能ではない。第二に、信頼の水準である。自己信頼、対人的信頼、一般化された信頼、制度信頼、政治的信頼、認識論的信頼、技術信頼は対象もメカニズムも異なる。第三に、信頼の時間性である。瞬間的状態、比較的安定した特性、関係史により厚みを持つ関係性、反復相互作用の中で更新される過程、文化や制度に埋め込まれた環境条件は区別が必要である。第四に、測定法の差である。質問紙は表明された期待や態度を、trust game はリスクを伴う委譲と返報期待を、神経画像やホルモン測定は構成要素過程を、面接やエスノグラフィーは意味づけと文脈を、微視的現象学的面接は経験構造を、それぞれ部分的にしか捉えない。第五に、負の概念との関係である。不信の欠如と mistrust、distrust、cynicism は同じではない。第六に、trust と trustworthiness の非対称性である。信頼することと、信頼されるに足ることは別概念であり、多くの論争は両者の混同から生じる。

神経現象学はこの地図の中で特異な位置にある。既存の信頼研究の多くは第三者的データに寄りがちで、信頼している当人にとっての経験の質感を粗くしか扱わない。他方、現象学だけでは行動や制度や神経過程との連結が弱い。神経現象学的アプローチは、このギャップを埋める候補である。すなわち、信頼を「監視の緩み」「身体の防御トーンの変化」「予測の委譲」「相手への可逆的な自己開示」「裏切られうることを知りつつ委ねること」といった経験次元に分解し、それを反復協力課題、相互行為観察、生理指標、脳活動、対話分析と結びつけることである。ただし、2026年3月時点で、信頼そのものを対象とした神経現象学研究はまだ限定的であり、現状では micro-phenomenology、愛着、安全感、社会的予測、認識論的信頼、相互行為の神経科学から慎重に橋を架ける段階にある。

本稿の暫定結論は単純である。信頼は一つではない。しかし無秩序でもない。広域に比較すると、信頼研究は「脆弱性」「期待」「依存」「予測不能性」「相手または制度の裁量」「監視の限界」「修復可能性」という共通構造を何らかの形で共有している。今後の深掘りでは、この共通構造を保ちながら、対人・制度・技術・自己・知識の各領域でどの要素が前景化するかを比較するのが有効である。

## 1. 分野別マップ

### 1.1 神経現象学（neurophenomenology）

- **定義**: 信頼そのものの定義は未定着。既存の神経現象学的手法から見ると、信頼は「他者や状況に身を預けられる感じ」「警戒がほどける感じ」「相手の不透明さに耐えつつ関与を続ける感じ」といった lived experience として扱いうる。
- **主な理論**: Varela 系の神経現象学、Petitmengin の micro-phenomenology、二人称的相互行為研究、enactive / embodied accounts。
- **主要な測定法**: micro-phenomenological interview、経験サンプリング、逐語分析、協力課題後インタビュー、行動指標・生理指標・神経データとの時系列対応。
- **代表的知見**: 信頼そのものを対象化した研究は少ないが、主観経験の微細記述は可能であり、経験の通時的・共時的構造を比較できる。相互行為の「接触感」「つながり感」が重要な質的基準となる。
- **典型的な論争点**: 想起された経験記述の信頼性、経験記述と神経データの対応単位、社会的文脈を失わずに経験構造を抽出できるか。
- **信頼を何として扱うか**: lived experience、身体化された過程、関係内の変容。

### 1.2 社会心理学

- **定義**: 他者が自分に損害を与えず、期待に沿ってふるまうだろうという対人的期待、あるいはその期待に基づく依存の意志。
- **主な理論**: 対人的期待モデル、信頼と協力の相互依存論、認知的信頼と情動的信頼の区別、propensity to trust と other-focused trust の区別。
- **主要な測定法**: Interpersonal Trust Scale 系質問紙、他者評価、シナリオ実験、反復協力課題、trust game。
- **代表的知見**: 信頼は相手の能力・善意・誠実性推定、評判、類似性、公平性知覚、約束や裏切り履歴により大きく変化する。
- **典型的な論争点**: 信頼は信念か態度か、表明された信頼と行動的信頼は一致するか、感情的成分をどこまで独立させるか。
- **信頼を何として扱うか**: 態度、期待、対人的評価、時に感情。

### 1.3 パーソナリティ心理学

- **定義**: 他者一般をどれほど信頼しやすいかという比較的安定した傾向、すなわち trustingness / propensity to trust。
- **主な理論**: disposition to trust、一般化された期待、Big Five や対人スタイルとの関連。
- **主要な測定法**: trait 尺度、人格検査、他者評価、縦断データ。
- **代表的知見**: 信頼傾向には個人差があり、低信頼者は脅威や否定的手がかりにより敏感である。高信頼は単純な楽観と一致しない。
- **典型的な論争点**: trait trust と一般化された社会的信頼を同一視してよいか、実際の相互行為による更新可能性をどこまで trait が拘束するか。
- **信頼を何として扱うか**: 特性、傾向、予期スタイル。

### 1.4 発達心理学

- **定義**: 乳幼児期の安全感から、学童期の selective trust、青年期以降の関係的信頼へと展開する発達的能力・志向性。
- **主な理論**: Erikson の basic trust、選択的信頼研究、心の理論と証言評価、社会的学習。
- **主要な測定法**: 養育観察、愛着評価、子どもの informant selection 課題、縦断研究。
- **代表的知見**: 子どもは早期から情報源の信頼性を選別する。年齢が上がると、単なる好ましさより知識や正確さを重視しやすくなる。
- **典型的な論争点**: basic trust を情緒的安全感として捉えるか、認識論的信頼の萌芽として捉えるか。社会的手がかりと認識論的手がかりの優先順位。
- **信頼を何として扱うか**: 発達課題、学習された期待、認識論的選別能力。

### 1.5 臨床心理学

- **定義**: 他者・自己・治療関係・情報源に対する安全感、開放性、あるいはその破綻。
- **主な理論**: 裏切りトラウマ、mentalizing と epistemic trust、治療同盟、愛着の臨床モデル。
- **主要な測定法**: 面接、症状尺度、治療同盟尺度、エピステミック・トラスト尺度、症例研究、縦断臨床研究。
- **代表的知見**: トラウマ、とくに裏切りを伴うトラウマは、対人信頼と制度信頼の破綻、解離、羞恥、警戒亢進に結びつく。治療では epistemic trust の回復が変化機序の一部と考えられる。
- **典型的な論争点**: trust の破綻を attachment のみで説明できるか、症状低減と trust repair は同じか、臨床でいう epistemic trust は哲学的 ET と同じか。
- **信頼を何として扱うか**: 安全感、対人的開放性、治療可能性の条件、修復可能な過程。

### 1.6 認知心理学

- **定義**: 他者・装置・情報源についての予測、学習、ヒューリスティック判断の結果としての trust。
- **主な理論**: event-based accounts、予測処理、社会的学習、情報源評価。
- **主要な測定法**: 課題遂行、反応時間、視線計測、反復ゲーム、計算モデル。
- **代表的知見**: trust は相手の行動履歴、文脈、報酬構造、予測誤差に応じて更新される。人はしばしば限定情報の下で信頼判断を行う。
- **典型的な論争点**: trust を高次の社会的意味づけと見るか、低次の学習・推論機構で十分説明できるか。
- **信頼を何として扱うか**: 学習・予測の結果、判断戦略、ヒューリスティック。

### 1.7 神経科学 / social neuroscience / neuroeconomics

- **定義**: 不確実な社会状況で相手に資源や裁量を委ねる行動、およびそれを支える神経・内分泌・学習機構。
- **主な理論**: trust game パラダイム、社会的価値計算、報酬学習、信頼と返報の神経モデル、large-scale brain network accounts。
- **主要な測定法**: trust game、反復投資ゲーム、fMRI、EEG/ERP、生理指標、ホルモン測定、遺伝関連研究。
- **代表的知見**: trust は単一脳部位ではなく、価値評価、社会予測、誤差学習、脅威処理、記憶、自己他者表象に関わる分散ネットワークで支えられる。trust と reciprocity は関連するが同一ではない。
- **典型的な論争点**: trust game が本当に trust を測っているか、リスク嗜好や社会的選好の混入、oxytocin 研究の再現性と一般化。
- **信頼を何として扱うか**: 行動選択、神経計算過程、学習と予測。

### 1.8 愛着理論・発達研究

- **定義**: 予測可能で応答的な養育者との関係から形成される「安全基地」としての基本的信頼。
- **主な理論**: Bowlby の internal working models、Ainsworth の愛着パターン、発達的安全感、養育者の可用性への期待。
- **主要な測定法**: Strange Situation、Adult Attachment Interview、縦断研究、養育観察。
- **代表的知見**: 一貫した応答性は安全感と他者への信頼可能性を育て、不安定・解離的養育は警戒、回避、両価性を強めうる。成人対人信頼や親密性にも波及する。
- **典型的な論争点**: 愛着と trust をどこまで同一視してよいか、成人の generalized trust まで直線的につながるか。
- **信頼を何として扱うか**: 関係的安全感、発達的基盤、内的作業モデル。

### 1.9 社会学

- **定義**: 社会的複雑性の縮減、一般化された期待、社会秩序を支える関係資源。particularized trust と generalized trust の区別が中心。
- **主な理論**: Luhmann、Simmel、Lewis & Weigert、Sztompka、社会資本論、制度埋め込み。
- **主要な測定法**: 社会調査、一般化信頼尺度、比較社会研究、ネットワーク分析、歴史社会学、民族誌。
- **代表的知見**: 特定他者への信頼と見知らぬ他者への一般化信頼は形成条件が異なる。信頼は個人心理だけでなく制度、規範、ネットワーク、格差に埋め込まれる。
- **典型的な論争点**: generalized trust は人格特性か制度成果か文化規範か、social capital と trust の因果方向。
- **信頼を何として扱うか**: 関係資源、社会規範、制度的条件、文化的期待。

### 1.10 政治学

- **定義**: 市民が政治主体や制度に対して「適切に統治し、規範に沿って行為する」と期待する関係。政治的 trust と trustworthiness を区別する。
- **主な理論**: Easton の support 理論、Levi & Stoker の relational account、制度パフォーマンス論、正統性論。
- **主要な測定法**: サーベイ、比較政治データ、測定等価性検証、パネルデータ、歴史比較。
- **代表的知見**: political trust は一枚岩ではなく、主体、対象、制度類型ごとに異なる。政府・議会・政党・警察・司法への trust は完全には同一因子ではない。
- **典型的な論争点**: trust と satisfaction と confidence の区別、低 trust は健全な批判性か民主主義危機か、distrust の政治的含意。
- **信頼を何として扱うか**: 制度への期待、正統性の一部、評価的態度、時に制度的資本。

### 1.11 経済学 / 行動経済学

- **定義**: 契約や監視で完全には保証できない相互行為のもとで、返報を期待して相手に資源を委ねること。
- **主な理論**: Berg らの trust game、reciprocity、social preferences、belief-based motivations、incomplete contracts。
- **主要な測定法**: trust game、gift exchange、public goods games、belief elicitation、構造推定。
- **代表的知見**: 人は単純な自己利益最大化から外れて trust と reciprocity を示す。ただし trust game の送金額には risk attitude や altruism が混入する。
- **典型的な論争点**: trust game は trust か risk か、返報性と分配選好をどう区別するか、外的妥当性。
- **信頼を何として扱うか**: 合理的選択、期待、戦略、社会的選好を含む行動。

### 1.12 組織論 / 組織心理 / マネジメント

- **定義**: 相手の行動を完全統制できない状況で、相手の能力・善意・誠実性などを踏まえて脆弱性を受け入れる意志。
- **主な理論**: Mayer, Davis, Schoorman の統合モデル、McAllister の cognition-based / affect-based trust、leadership trust、trust repair。
- **主要な測定法**: 職場調査、上司-部下尺度、ネットワーク調査、フィールド研究、メタ分析。
- **代表的知見**: trust は job satisfaction や commitment との関連が強い。認知的 trust と情動的 trust は区別可能で、関係の深さや役割に応じて働き方が異なる。
- **典型的な論争点**: trust と trustworthiness の測定混同、個人間 trust と組織 trust の水準混同、violation 後の repair 条件。
- **信頼を何として扱うか**: 態度、関係の質、脆弱性受容、協働資源。

### 1.13 哲学（倫理学・社会認識論・現象学）

- **定義**: 他者の善意・理由・規範的応答性への依存。多くの議論で reliance との差異、vulnerability、betrayal possibility が中心。
- **主な理論**: Baier の vulnerability account、Hardin の encapsulated interest、motives-based theories、reliance plus account、epistemic trust in testimony、self-trust。
- **主要な測定法**: 概念分析、規範理論、事例分析。経験的測定より概念区別が中心。
- **代表的知見**: 信頼は mere reliance ではなく、裏切りの可能性や相手の動機への期待を含むとされることが多い。信頼と trustworthiness は別概念であり、不信は trust の不在ではない。
- **典型的な論争点**: trust の「追加要素」は何か。善意か、規範的期待か、自己利益の包摂か。epistemic trust は testimony への reliance とどう違うか。
- **信頼を何として扱うか**: 規範的関係、態度、脆弱性の受容、認識論的スタンス。

### 1.14 人類学 / 民族誌

- **定義**: 抽象的一般概念というより、具体的な関係実践、道徳秩序、相互扶助、疑念、儀礼、フィールドワーク内の関係構築として把握される。
- **主な理論**: ethnographic trust-building、mistrust as stance、地域的・文化的文脈に埋め込まれた trust/distrust。
- **主要な測定法**: 長期参与観察、インタビュー、フィールドノート、厚い記述。
- **代表的知見**: trust はしばしば均質な「善」ではなく、関係上の義務、沈黙、恥、噂、疑念、ケア実践の中に現れる。mistrust も単なる欠如ではなく、社会を組織する正当な姿勢たりうる。
- **典型的な論争点**: trust を普遍概念として輸出してよいか、民族誌の信頼性をどう担保するか、研究者-参加者関係そのものが trust 問題を含むこと。
- **信頼を何として扱うか**: 関係実践、道徳秩序、文脈依存の lived relation。

### 1.15 公衆衛生 / 医療 / 科学への信頼

- **定義**: 医師、医療機関、保健システム、科学者、科学的方法、エビデンス、保健勧告への期待と依存。
- **主な理論**: physician trust、health system trust、medical mistrust、public trust in science、science communication、institutional trust ecosystems。
- **主要な測定法**: 尺度開発、患者調査、システム信頼尺度、ワクチン・医療受療関連研究、質的研究、体系的レビュー。
- **代表的知見**: competence だけでなく integrity、公正さ、透明性、利益相反、歴史的不正義が trust に強く影響する。医療では特に mistrust が健康格差と結びつく。
- **典型的な論争点**: trust in physician と trust in health system の区別、科学コミュニケーションの質と制度的信頼の区別、健全な懐疑と有害な不信の境界。
- **信頼を何として扱うか**: 制度的・認識論的信頼、ケア関係、正統性、歴史的記憶。

### 1.16 HCI / trust in automation / AI への信頼

- **定義**: 自動化・AI に対してどれだけ依存するかを方向づける心理状態。適切な reliance と calibration が中心課題。
- **主な理論**: Lee & See の appropriate reliance、Hoff & Bashir の three-layer model、calibrated trust、trust in automation measurement、HCI trust frameworks。
- **主要な測定法**: self-report trust scales、行動的 reliance 指標、パフォーマンス指標、故障後の trust repair 実験、UX 調査。
- **代表的知見**: trust はシステム性能だけでなく、説明可能性、透明性、エラー経験、擬人化、ユーザ特性、タスク文脈により変動する。高 trust が常に望ましいわけではなく、overtrust / undertrust が問題。
- **典型的な論争点**: 人への trust を機械にそのまま移植できるか、trust と reliance を区別すべきか、AI trustworthiness を perception だけで代理してよいか。
- **信頼を何として扱うか**: 期待、校正された依存、状況的判断、設計対象。

### 1.17 法学（補足）

- **定義**: 裁判所、警察、法制度への trust、あるいは逆に legal cynicism を通じた法への不信。
- **主な理論**: legitimacy と trust の区別、procedural justice、legal cynicism。
- **主要な測定法**: サーベイ、質的研究、制度比較。
- **代表的知見**: 制度への trust は単なる効率期待だけでなく、公正性と尊重の経験に強く依存する。
- **典型的な論争点**: trust と legitimacy の因果関係、法制度不信が抵抗か逸脱か。
- **信頼を何として扱うか**: 制度的期待、正統性評価。

### 1.18 教育学（補足）

- **定義**: 学校共同体内の relational trust。教師、保護者、校長、生徒の相互期待。
- **主な理論**: Bryk & Schneider の relational trust。
- **主要な測定法**: 学校調査、質的研究、改革追跡。
- **代表的知見**: respect、competence、personal regard、integrity が school improvement の資源になる。
- **典型的な論争点**: trust を文化資本や学校資源からどこまで独立に扱えるか。
- **信頼を何として扱うか**: 組織的関係資源、改善能力の条件。

### 1.19 宗教学 / 宗教研究（補足）

- **定義**: 宗教的帰属や religiosity が対人的 trust や institutional trust にどう関わるか。
- **主な理論**: moral community、宗教的アイデンティティと内集団 trust、超越的監視仮説に近い議論。
- **主要な測定法**: survey、trust game、比較文化実験。
- **代表的知見**: 宗教は一般化 trust を高める場合も、内集団偏重を強める場合もあり、効果は文脈依存。
- **典型的な論争点**: 宗教が普遍的 trust を育てるのか、境界線を強めるのか。
- **信頼を何として扱うか**: 規範、共同体帰属、信念と相互期待。

### 1.20 進化論 / 進化心理学（補足）

- **定義**: 協力・返報・評判・社会的監視の進化の中で維持される信頼と trustworthiness。
- **主な理論**: 返報性、評判、社会的監視、進化ゲーム。
- **主要な測定法**: 進化モデル、比較行動研究、実験ゲーム。
- **代表的知見**: trust は無条件善意ではなく、繰り返し相互作用、監視可能性、評判情報、個体差と結びつく。
- **典型的な論争点**: 人間特有性、社会規範と生物学的適応の比重。
- **信頼を何として扱うか**: 適応的戦略、協力維持機構。

### 1.21 情報システム論（補足）

- **定義**: 不確実なオンライン環境で、制度的保証と相手への belief に基づいて行う委譲。
- **主な理論**: disposition to trust、institution-based trust、trusting beliefs、trusting intentions。
- **主要な測定法**: e-commerce 調査、構造方程式モデル、初期 trust 形成モデル。
- **代表的知見**: 対面経験がなくても、制度的保証、評判、インターフェース、既有傾向により initial trust は形成されうる。
- **典型的な論争点**: オンライン trust は対人 trust の派生か独自構成概念か。
- **信頼を何として扱うか**: 制度的保証、認知的信念、行動意図。

## 3. 神経現象学と心理学の位置づけ

### 3.1 なぜ神経現象学を独立に立てる必要があるか

既存の信頼研究は、質問紙と行動課題に大きく依存している。これにより、「誰をどれくらい信頼するか」「どれだけ資源を委ねたか」は捉えやすいが、「信頼している感じ」がどのような経験構造をもつかは粗くしか見えない。たとえば trust game で高額送金したとしても、そのときの体験は一様ではない。楽観的な開放、戦略的計算、相手への賭け、諦め、関係形成の誘い、あるいは緊張したままの委譲は区別されうる。神経現象学の必要性は、この差を「行動の背後」に還元せず、経験構造そのものとして記述する点にある。

### 3.2 神経現象学で信頼をどう扱いうるか

信頼の主観的経験を研究する場合、少なくとも以下の経験次元が候補になる。

- **警戒のトーン**: 身体がどれだけ硬いか、相手を監視し続けている感じがあるか。
- **委譲感**: 自分で完全制御せず、相手や状況に一部を任せた感じがあるか。
- **可逆性の感覚**: 「まだ引き返せる」「もう身を預けた」の境界がどこにあるか。
- **相手の不透明さへの耐性**: 相手の意図が読めなくても関与を続けられるか。
- **身体感覚**: 胸・腹・喉・顔・姿勢・呼吸などにどのような変化が起きるか。
- **時間性**: 信頼が一瞬で立ち上がるのか、反復を通じて厚みを持つのか。
- **自己感**: 信頼時に自分が拡張されるのか、脆く露出するのか、支えられるのか。

これらは既存の社会心理学尺度では十分に切り出しにくいが、micro-phenomenological interview であれば、特定の瞬間を evocation させつつ、通時的連鎖と同時的構造に分けて記述できる。

### 3.3 心理学全般との接続

- **愛着研究との接続**: 信頼経験の土台には「相手は戻ってくる」「見捨てられない」という予測可能性がある。これは愛着研究の internal working models と接続する。
- **社会認知との接続**: 信頼は他者の意図推定、心の理論、評判処理、約束の理解と結びつく。
- **感情研究との接続**: 安堵、安心、好意、希望、羞恥、恐怖、裏切りの痛みは trust dynamics の一部だが、信頼全体を感情に還元することはできない。
- **自己感との接続**: 信頼は自己境界を一時的に開く。自己効力感や self-trust と連動する場合もあれば、自己の脆弱性の受容として現れる場合もある。
- **身体感覚との接続**: trust はしばしば threat monitoring の低下や、姿勢・視線・呼吸・声の変化として現れる可能性がある。

### 3.4 1人称報告と3人称データをどう結ぶか

信頼の神経現象学的研究プログラムとしては、以下のような設計が考えられる。

1. 反復協力課題や自然場面に近い dyadic task を実施する。  
2. 各試行または特徴的エピソード後に短い経験サンプリングを行う。  
3. セッション後に micro-phenomenological interview で特定瞬間を詳細に再構成する。  
4. 行動指標、反応時間、表情・視線、心拍変動、皮膚電気、生理指標、必要に応じて EEG/fMRI と対応づける。  
5. 経験カテゴリを trial-by-trial の学習モデルや相互行為の時系列にマッピングする。  

このとき重要なのは、脳活動を「信頼の部位」に還元しないことである。現実には、信頼経験は予測、価値、身体調整、社会的意味づけの結節点として生じる。神経データは lived experience の代替ではなく、その制約条件や随伴過程を示す第三者的記述として位置づけるべきである。

### 3.5 既存研究が少ない場合にどこまで言えるか

2026-03-26 時点で、信頼そのものを主題とした神経現象学研究は未整理であり、主要文献は method-centered である。したがって、以下は近接領域からの慎重な推定である。

- 信頼の経験は、他者予測の確率評価だけでなく、身体化された安全感の変化を含む可能性が高い。
- 裏切り後の trust repair は、認知的再評価だけでなく、再び「監視をほどくことができるか」という経験的閾値の問題である。
- 技術信頼や制度信頼にも lived dimension は存在するが、対人信頼より抽象度が高く、身体経験としては間接的に現れる可能性がある。

## 4. 主要論争

### 4.1 信頼は感情か、信念か、期待か、関係か

分野ごとに答えは違う。社会心理学と組織論の一部では、信頼は肯定的期待や脆弱性受容の意志として定式化される。哲学では、期待に加えて相手の動機や裏切り可能性が問題になる。社会学では、個人内状態よりも、社会関係や制度に埋め込まれた一般化期待として扱われる。HCI では reliance を媒介する心理状態として定義されやすい。したがって「信頼の本質」を一つに還元するより、「どのレベルの現象を問題にしているか」を先に確定する方がよい。

### 4.2 信頼とリスクの関係

多くの分野で、信頼は不確実性や vulnerability を前提とする。ただし、経済学では risk が明示的選択問題として扱われやすいのに対し、哲学では risk は裏切り可能性と規範性に結びつき、臨床では傷つけられうることへの身体的・情緒的開放を意味する。trust game に対する主要な批判もここにあり、そこで測られるものは trust だけでなく risk tolerance でもある。

### 4.3 信頼と脆弱性（vulnerability）

Baier 以降、信頼を vulnerability から切り離せないとする議論は強い。だが、HCI や一部の情報システム研究では、この vulnerability は「完全理解や完全監視ができないので依存する」という形で現れ、人間への moral vulnerability とは異なる。ゆえに、脆弱性は共通構造であっても、その内容は対象により変わる。

### 4.4 信頼と裏切り・修復

裏切りは trust 特有の損傷であり、mere reliance の失敗とは異なる。組織論、臨床、対人関係研究は、修復には単なる説明だけでなく、責任承認、再発防止、時間、反復的一貫性が必要だと示している。神経現象学的には、修復は「もう一度、警戒を解けるか」の問題として捉え直せる。

### 4.5 信頼と制度・権力

社会学、政治学、医療、法学では trust は権力と不可分である。市民や患者が制度を trust するのは、制度が強制力や専門性を持つからであり、同時にその trust は歴史的不正義や排除によって損なわれうる。したがって制度信頼は「個人がもっと信じればよい」という問題ではなく、制度側の trustworthiness と accountability の問題でもある。

### 4.6 信頼と不信の違い

ほぼすべての分野で、distrust は trust のゼロ点ではない。哲学では distrust は積極的な否定的期待であり、政治学では cynical distrust と healthy skepticism は区別される。人類学でも mistrust は世界への適切な構えでありうる。よって low trust をそのまま欠損や病理として扱うのは危険である。

### 4.7 信頼と trustworthiness の違い

trust は trustor 側の態度・期待・行動、trustworthiness は trustee 側の性質・能力・動機・制度設計である。組織論では ability / benevolence / integrity が trustworthiness の主要成分とされ、政治学では制度の performance と fairness、医療では competence と integrity、AI では reliability と transparency が対応する。両者の混同は測定上の大きな問題である。

### 4.8 人間への信頼と AI / 自動化への信頼は同じか

完全には同じではない。AI には善意や道徳的責任主体性を帰属しない立場も強く、その場合 AI trust は実質的には calibrated reliance となる。ただし実際のユーザは擬人化し、社会的手がかりで AI を判断するため、対人信頼のメカニズムが部分的に流入する。したがって「同じ」でも「全く別」でもなく、部分重複モデルが最も妥当である。

## 6. 深掘り候補

1. **愛着と信頼**  
   basic trust、愛着安全性、成人対人信頼、自己信頼の連結を詰める。

2. **信頼の神経現象学**  
   「警戒がほどける感じ」「身を預ける感じ」を micro-phenomenology と生理・行動データでどう結ぶか。

3. **trust game の限界**  
   trust / risk / reciprocity / altruism の混入をどこまで分離できるか。

4. **政治的信頼と制度信頼**  
   政治主体への不信と国家制度への信頼がどう分岐するか。

5. **科学・医療への信頼**  
   competence, integrity, fairness, historical injustice をどう切り分けるか。

6. **AI への信頼と reliance**  
   人間信頼との連続性、擬人化、校正、trustworthiness 指標の整理。

7. **信頼の修復**  
   対人、組織、制度、医療、AI で repair 条件がどう異なるか。

8. **認識論的信頼**  
   testimony、専門家、子どもの selective trust、臨床の epistemic trust を横断比較する。

9. **不信・mistrust・distrust の地図**  
   不信を単なる欠如ではなく、どの分野でどう積極概念化しているかを見る。

10. **自己信頼**  
   self-trust を臨床、フェミニスト哲学、自己効力感研究とつなげる。

- **態度としての信頼**: 社会心理学、政治学、医療 survey、HCI の self-report。
- **期待としての信頼**: 社会心理学、経済学、政治学、情報システム論。
- **関係の性質としての信頼**: 愛着、組織、教育、民族誌。
- **社会規範としての信頼**: 社会学、宗教研究、政治文化研究。
- **制度的資本としての信頼**: 社会学、政治学、公衆衛生、法学。
- **合理的選択 / リスク判断としての信頼**: 経済学、HCI、情報システム論。
- **学習・予測の結果としての信頼**: 認知心理学、神経科学、発達心理学。
- **lived experience としての信頼**: 神経現象学、現象学、臨床、民族誌。

## 付記: 測定法ごとの「何を測れていて、何を測れていないか」

- **質問紙**: 態度・表明された期待・一般傾向は測りやすい。だが状況依存の揺れ、身体経験、相互行為の時間性は弱い。
- **trust game / economic games**: 委譲行動と返報性を可視化できる。だが risk preference、social preferences、実験理解が混入する。
- **実験**: 因果推定に強い。だが生態学的妥当性が限定されやすい。
- **縦断研究**: 形成と更新を追える。だが測定概念の安定性が課題。
- **面接 / エスノグラフィー**: 文脈・意味・修復・権力を掴みやすい。だが比較標準化は難しい。
- **微視的現象学的インタビュー**: 経験構造を詳細に記述できる。だが訓練コストと分析負荷が高い。
- **生理指標**: 覚醒や安全化の身体側面に迫れる。だがそれ自体は trust の同定ではない。
- **神経画像**: 構成要素過程の分解に有効。だが「信頼中枢」を探す発想は単純すぎる。
- **ホルモン測定**: 社会的接近やストレス調整との関連を見る補助になる。だが一般化しすぎると誤る。
