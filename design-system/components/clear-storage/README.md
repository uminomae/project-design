# clear-storage

dev utility — クリックで localStorage / sessionStorage を全消去し、トップへリダイレクト。

## 使い方

各 repo の root に配置 (or pd 参照):

```
{repo}/clear-storage.html → pd design-system/components/clear-storage/index.html
```

footer 等から `<a href="./clear-storage.html?dev">clear storage</a>` でリンクするだけ。

## 依存

なし (token / class / framework すべて不要)。

## 出自

cs/clear-storage.html (techo#129 Phase 2-A で pd 集約、pd#92)。
