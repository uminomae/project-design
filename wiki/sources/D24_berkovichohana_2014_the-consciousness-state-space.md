# 意識状態空間（CSS）— 意識と自己の統合モデル

> **高校生向けのやさしい解説**
>
> 意識のあり方を「時間（今この瞬間〜過去未来の想起）」「気づき（無自覚〜明瞭）」「情動（覚醒度と快不快）」の 3 次元の地図に整理した理論的論文です。日常意識・フロー・瞑想・夢などのさまざまな意識状態を、この『意識状態空間（CSS）』の中に位置づけられる。「今この瞬間の core consciousness」と「物語的な extended consciousness」が脳の中で互いを抑制しあい、瞑想ではこの抑制が緩む——という仮説を含みます。

## 概要

本理論モデルは、意識と自己が 3 つの現象学的次元——時間、気づき、情動——の相互作用から立ち上がると提唱する。著者らは現在瞬間における最小自己性を支える core consciousness（CC）と、時間を超えた物語的同一性を支える extended consciousness（EC）を区別し、これらを神経ネットワーク（intrinsic / extrinsic）にマップする。フローや瞑想などの altered states も同枠組みで分析される。

## 主要概念

**CSS の枠組み — 身体を中心とした 2 つの同心球**

内側の球：直接的感覚経験と即時的主体性。
外側の球：概念的思考、記憶、自伝的同一性。

これらは同時的だが通常は拮抗的な経験軌道として機能し、意識的気づきへのアクセスが切り替わる。

**3 次元 — 時間・気づき・情動**

**1. 時間**：現在瞬間（< 3 秒）の即時知覚から、過去記憶と未来予測を含むメンタル・タイムトラベルまで。「Time is an omni-present structural feature of consciousness」。

**2. 気づき**：完全な無意識から閾下処理を経て高度な現象アクセスまで。アクセス意識（注意を要する報告可能経験）と現象意識（一人称私的経験）を区別。

**3. 情動**：覚醒（身体的興奮）と価（精神的快不快）の 2 成分。独立に変動し、異なる意識アクセス度合いで体験できる。

**意識の根本性としての時間**

> "consciousness would be inconceivable without temporality"

時間性はあらゆる経験に根本的。

**Intrinsic / Extrinsic ネットワークの拮抗**

脳は intrinsic（自己参照的）と extrinsic（外向き）ネットワークを交互に使う。通常は対立的だが、altered self states ではこの拮抗が低下する。

**フローと瞑想の共通構造**

> "a transition toward the CC/MS trajectory being more available to access awareness"

両状態は核心意識/最小自己軌道がアクセス気づきにより利用可能になる、自己参照処理の低下と『典型的に拮抗的なネットワーク間の同期増加』を伴う。

**統合的説明力**

> "CSS provides a phenomenological map which includes all possible consciousness states"

意識・自己・時間・情動についての従来別個の理論的説明を、神経科学的証拠に基づく一貫した建築の中に統合する。

## 方法

理論論文。現象学的分析と神経科学的証拠の統合。Damasio の core/extended consciousness 区別、Default Mode Network 研究、瞑想・フロー研究を統合。
