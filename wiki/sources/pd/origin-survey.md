# 30 学問領域「原点」調査

> **高校生向けのやさしい解説**
>
> ふだん見慣れた「0」（ゼロ）。数学では加算の単位元ですが、物理では「絶対零度」や「真空」、生物では「絶滅」や「アビオジェネシス」、法学では「テラ・ヌリウス（無主地）」、哲学ではパルメニデスの「無は語れない」というように、学問領域ごとに「ゼロ・原点・無・空・欠如」の扱いはまったく違います。この調査は 30 の学問領域でそれらの概念を並べ、「否定的に扱われるか」「肯定的に扱われるか」「権力的に使われるか（支配を正当化する装置になっていないか）」の 3 軸で比較した PD 独自の横断調査です。結論は「単一の原点はない」。30 領域に共通する『弱い構造』として『境界装置』『語りの限界』『変化の知覚閾値』が浮かぶ一方、権力的ゼロ（Year Zero、テラ・ヌリウス、社会的死）など『足場』の逆機能も 7 件以上確認されました。

## 概要

本調査は、PD が 30 の学問領域それぞれで「ゼロ・原点・無・空・欠如」に相当する概念を同一フォーマットで調査・比較する横断調査である。2 つの LLM（Claude Opus 4.6 / Codex GPT-5）による並行調査と、v1 → v2 → v3 の 3 回の反復を経て実施された。v1 は仮説主導で偏りが出たため、v2 でプロトコルを「事実収集」と「仮説突き合わせ」の 2 段階に分離。v3 で「権力的ゼロ」「語りの限界」「見つからなかった」の許容を追加した。

最終的な信頼度は**中（部分的支持）**。「30 領域に強い共通構造がある」という強い仮説は退けられ、「多くの領域で何らかの役割を持つ」という弱い版のみが支持される。最も頑健に浮上したのは「各領域の『ゼロ』は静的な点ではなく **変化が知覚可能になる閾値（境界装置）** として機能する」「各領域ごとに固有の **『語り方の限界』** が露呈する」の 2 点である。反対に、権力的ゼロ（Year Zero、テラ・ヌリウス、社会的死、優生学、MKUltra、capitis deminutio、文化大革命）は 7 件以上確認され、「ゼロが生成の足場になる」仮説への強い反証群となっている。

**3 軸評価フォーマット（各領域共通）**

各領域について、概念ごとに以下の 3-4 軸で記述する:

- **否定的な扱い**: 「〜がない / 足りない / 到達不能」等の欠如として扱われる側面
- **肯定的な扱い**: 基準点・体系の前提・生成条件として機能する側面
- **権力的な扱い**（v3 で追加）: ゼロ宣言が暴力・支配・排除の装置として作動する事例
- **関係類型**: 否定と肯定の関係（同時的共存 / 段階的変容 / レベル差 / 学派依存 / 文化依存 / 片方のみ）
- **語りの限界**: その領域でゼロについて語るときに露呈する言語化の限界

**主要な発見（3 つの強い共通構造）**

1. **「境界装置」（boundary device）としてのゼロ**: 30 領域で独立に浮上。Codex は 4 群分類（基準点型 / 生成的空白型 / 否定的開示型 / 病理・破綻型）、Claude は 3 性質抽出（測定可能性の条件 / 生成の条件 / 到達不能性）で到達。pjdhiro の「赤外線/赤の境界」アナロジーと併せて 3 者で独立収束。

2. **「変化の知覚閾値」としてのゼロ**: D02 絶対零度・D04 絶滅・D08 resting potential・D13 パルメニデス・D18 anomie・D25 liminality・D29 criticality が共通して「変化が知覚可能になる閾値」として機能する。静的な点ではなく、差分・転換・境界を生む働き。

3. **「語りの限界」の共有**: D13 パルメニデス（無について語ることは不可能）、D24 否定神学（神は〜でない、としか言えない）、D17 ゼロ形態素（ないのにあるかのように扱う）、D19 negative capability（答えが出ない状態に耐える）、D06 特異点（物理法則が破綻し記述不能）— 各領域は「その領域固有の語れなくなる地点の構造」を持つ。

**強い反証: 権力的ゼロ（7 件以上）**

「ゼロが生成の足場になる」仮説への反証として、ゼロの宣言が存在抹消の装置として機能する事例群が独立に確認された:

- **D16** Year Zero（カンボジア、人口の約 25% が死亡）
- **D20** テラ・ヌリウス（204 年間の法的不可視化）
- **D20** 法的人格剥奪（capitis deminutio、ニュルンベルク法）
- **D18** 社会的死（奴隷制による社会的ゼロ化）
- **D08** MKUltra（感覚遮断ゼロ状態を洗脳に使用）
- **D04** 優生学（「不適者」の繁殖をゼロ化）
- **D16** 文化大革命、ソビエト暦、イラン革命（構造的に反復する「歴史ゼロ化」パターン）

内部で 2 サブタイプに分かれる:
- **消去型**: 存在するものをゼロと宣言して抹消（テラ・ヌリウス、Year Zero、法的人格剥奪）
- **強制型**: 他者に自己をゼロにさせる（MKUltra、ヴェイユの減創転用）

**「語りの限界」の 3 種**

構造的類似はあるが、内部で 3 種に分かれる:

1. **再帰的不可能性**（到達すれば報告不能）: D13 ヴェイユ、D24 ファナー、D19 ベケット
2. **二項対立の崩壊**（語ると対象が変質）: D14 無意識、D15 不在の美、D19 沈黙
3. **境界線の流動性**（ゼロ地点を確定できない）: D04 生命/非生命、D23 発達のゼロ地点、D06 t=0

## 方法

**3 段階反復（v1 → v2 → v3）**

| Version | 主な変更 | 問題点と改善 |
|---|---|---|
| v1 | 仮説主導「ゼロは有るが立ち上がる足場か」で Stage 1 を走らせた | 30/30 で仮説支持の結果、バイアス重大 |
| v2 | 仮説を Stage 1 から排除、Stage 1=事実収集 / Stage 2=突き合わせに分離 | 否定的扱いが 0/30 → 30/30 に改善 |
| v3 | 「見つからなかった」の許容、権力的ゼロ軸の追加、語りの限界の明示、5 類型の明示 | 「テラ・ヌリウス型」反証群が体系化された |

**並行モデル戦略（v2）**

v2 で Claude Opus 4.6 と Codex GPT-5 による並行調査を実施し、クロスモデルレビューで事実由来の収束とモデル固有バイアスを分離した。

- **事実由来の収束（高信頼）**: D13 哲学の内部多様性、D29 の「境界性」、30/30 での否定的扱い、「単一収束しない」という結論
- **プロンプト由来の収束（バイアスの可能性）**: 30/30 で「肯定的扱いもあり」になる圧力、「ゼロ的概念を探せ」という選択バイアス
- **モデル固有の偏り**: Codex=簡潔で反証漏れ / Claude=網羅的だが「同時的」判定過剰、文化圏カバーは Claude が厚い

v3 では費用対効果を考え 1 モデル + 人間レビューに切り替え。

**Stage 3 吟味レポート（自己評価）**

各モデルは Stage 3 で自作レポートを 3 観点（事実精度 / 構造的偏り / 三角測量の有効性）で評価。Claude v3 の自己評価は全項目 **B** 相当。

## 結論と信頼度

**最終判定（v3）: 信頼度「中（部分的支持）」**

- **弱い版「ゼロ概念が多くの領域で体系の成立に関与する何らかの役割を持つ」**: 中程度で支持
- **強い版「30 領域に共通構造があり原点が足場として機能する」**: 低い信頼度

**次に調べるべきこと**（優先順）:
1. 問いを「各領域のゼロ概念は何か」→「各領域で変化が知覚可能になる閾値はどこか」に転換した v4 設計
2. 権力的ゼロ（テラ・ヌリウス型）の単独深掘り（D16/D18/D20/D25 横断）
3. 「語りの限界」3 種それぞれの単独分析
4. 判定不能（E1-E5）の個別精査

## プロジェクトデザインとの関連

本調査は PD が外部理論を輸入する前に「ゼロ・無・欠如」という PD 論の中核と接する概念を 30 領域で横断的に把握するための基礎作業である。具体的には:

- **欠損駆動思考との接続**: PD の「欠損（あってほしいのに、ない）」と各領域の「ゼロ・欠如」概念の対応を取るとき、本調査の「境界装置」「変化の知覚閾値」の視点は PD の欠損が『単なる空白』ではなく『差分を生む境界』であることを補強する
- **権力的ゼロの発見**: ゼロの宣言が暴力になりうるという知見は、PD のデザイン論における「欠損をどう扱うか」の倫理的次元を明示する。形骸化議論（デザイン思考論）との接続可能性
- **「語りの限界」の 3 分類**: PD の「保持論点を急いで解くな」原則と共鳴する。1 人称記述の限界、到達すれば報告不能な経験構造、境界線の流動性
- **30 領域のマップ**: PD が今後どの領域と深く接続するかの優先順位判断の基礎データ
