# 信頼の多元的記述

## 定義

信頼を単一の定義で捉えるのではなく、複数の独立した軸から多元的に指し示す分析フレームワーク。[[信頼の定義仮説]]（CN-005）の仮説群を評価・整理するためのレンズとして機能する。「期待」と「信頼」の日常語としての混同を解消し、生存-信頼軸と時間軸が独立であることを示す。

## 理論的背景

Session 17 での ISS-41-a の議論から派生した。H01-H04（精神分析的視点の仮説群）と H05-H08（生存-信頼軸の仮説群）の関係を検討する中で、「上位の状態記述と下位の機能記述の包含関係」が見え、さらに pjdhiro から「期待と信頼の違い」というヒントが提示された。議論の結果、期待-信頼軸は生存-信頼軸と独立であることが判明し、ラベルではなく軸で多元的に記述する方針が採用された。

Klein の PS ポジション/D ポジションが内容軸の理論的根拠を、Bowlby の愛着理論が時間軸（蓄積としての信頼）の根拠を、神経科学の知見（vmPFC-扁桃体系の構造的接続 vs 一時的活性化パターン）が状態軸の根拠を提供する。

## 5つの記述軸

1. **時間軸**: 未来的（期待的、予測、不確実）と過去-現在的（信頼的、蓄積、確定）の対比
2. **内容軸**: 良い結果のみ（分裂的、期待的）と良いも悪いも（統合的、信頼的）の対比
3. **状態軸**: 活動パターン（一時的、H08.pl）と構造的接続（持続的、H08.bs）の対比。Session 18 の H08 二成分モデルに直接対応する
4. **生存-信頼軸との直交性**: 時間軸・内容軸は生存-信頼軸と直交する。生存軸にも信頼的状態がありうる（物理法則への信頼）
5. **循環軸**（Session 18 追加）: 個人内循環（Secure Base 経験と H08 と Secure Base 提供能力の相互作用）および世代間螺旋（時間方向に積み重なるネスト構造）

## 会計メタファとの接続

期待的なものは「未実現」（予算、含み益）、信頼的なものは「実現済み蓄積」（確定純資産）に対応する。ただし PL=期待、BS=信頼という単純対応ではなく、「未実現 vs 実現済み蓄積」が本質。PL にも実績があり、BS にも含み（未実現）がある。
