# アブダクション

## 概要

アブダクション (abduction) は、C. S. Peirce が提唱した第三の推論形式であり、演繹 (deduction) や帰納 (induction) とは異なり、「驚き・違和感から説明仮説を形成する」論理操作である。Peirce はこれを「新しいアイデアを導入する唯一の論理操作」と位置づけた。[[デザイン思考]]の文脈では Kees Dorst (2011) が abduction を設計実践に接続し、framing / frame creation の核心操作として精緻化した。

## 本プロジェクトとの関連

アブダクションは本プロジェクトの中核概念である[[欠損駆動思考]]と直接接続する。[[PDブリッジ保持論点]]において、科学のアブダクションが既存世界の説明仮説を立てるのに対し、design のアブダクションは「まだ存在しないが望ましいもの」を扱うとされ、欠損駆動思考は後者に近い -- 棄却された誤差から「あるべきだが、ない」を読み取る態度である。

ただし欠損駆動思考はアブダクションそのものではなく、アブダクションを可能にする**前態度**として位置づけられている。欠損駆動思考（態度）が先に作動し、その後にアブダクション（仮説生成）が続く。

### 設計固有のアブダクション

Dorst (2011) の frame creation は Peirce 的アブダクションの設計的拡張として理解できるが、完全同一視は言い過ぎである。Frame creation は説明仮説だけでなく、価値判断・境界設定・評価軸の設計を含み、外化（プロトタイプ）を通じて前進する点で言語命題の推論に還元しにくい。

## 主要な知見・引用

- **3推論形式の比較**: 演繹（既知ルール → 結論）、帰納（事例 → 一般化）、アブダクション（驚き → 説明仮説）
- **Design framing との関係**: Dorst (2011) は design thinking の核心を framing / frame creation に求め、アブダクションとの関係を精緻化
- **深掘り候補**: 「科学のアブダクションと design のアブダクションはどこで分岐するか」は統合レポートの未決問い
