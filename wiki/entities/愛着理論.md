# 愛着理論

## 概要

愛着理論 (Attachment Theory) は John Bowlby (1907-1990) が提唱し、Mary Ainsworth が Strange Situation 実験で実証的に展開した、養育者との関係から形成される安全基地 (Secure Base) の理論。養育者の応答性への期待から内的作業モデル (Internal Working Model) が形成され、それが成人期の対人関係や信頼の基盤となる。Erik Erikson の basic trust（基本的信頼）の概念とも接続する。

## 本プロジェクトとの関連

愛着理論は本プロジェクトの複数の概念と深く接続する。

[[信頼の定義仮説]]において、H01-H04（精神分析的視点の仮説群）の理論的基盤の一つであり、特に Bowlby の安全基地概念は H01-H04 が「状態層」として信頼の構造的基盤を記述する根拠を提供する。

[[信頼の多元的記述]]では、時間軸（蓄積としての信頼）の理論的根拠として愛着理論が位置づけられている。また、循環軸（Session 18 追加）において、Secure Base 経験と H08 と Secure Base 提供能力の相互作用という個人内循環、および世代間螺旋の記述に愛着理論の枠組みが用いられている。

[[測定設計原則]]の P2（同一回路・二時間窓）では、bs 成分の測定に AAI（Adult Attachment Interview）が代表的手法として挙げられている。

## 主要な知見・引用

- **安全基地 (Secure Base)**: 養育者の応答性への期待から形成される。探索行動の拠点として機能する
- **内的作業モデル**: 初期の愛着関係から形成される自己と他者に関する表象。成人の対人関係パターンを規定する
- **愛着パターン**: Ainsworth の分類（安定型、回避型、両価型）および後の無秩序型
- **Basic trust** (Erikson): 発達初期の基本的信頼の形成が、後の信頼能力の基盤となる
