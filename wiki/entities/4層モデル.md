# 4層モデル

意識を身体状態から反応の見直しまでの流れとして捉える構造モデル。[[内受容感覚]]（Layer 0）、予測-誤差ループ（Layer 1）、生存-信頼評価（Layer 2）、[[抱持]]（Layer 3）の4層からなる。

## 概要

4層モデルは、意識を静的な内容の集まりとしてではなく、作動の流れとして記述するためのモデルである。身体状態、差の検出、方向づけ、見直しの余地という順序で意識を読む。

このモデルの利点は、同じ「つらさ」や「焦り」を一つの塊として扱わずに済むことにある。身体状態の揺れなのか、差の前景化なのか、方向づけの問題なのか、見直す余地を失っているのかを、段階ごとに読み分けられる。

## 4層の構造

| Layer | 名称 | 機能 | 神経基盤 |
|-------|------|------|---------|
| **Layer 0** | [[内受容感覚]] | 身体状態のモニタリング | 島皮質 |
| **Layer 1** | 予測-誤差ループ | 誤差検出、[[欠損]]生成 | 皮質階層 |
| **Layer 2** | 生存-信頼評価 | 価値判断、[[情動の構成]] | 扁桃体、vmPFC |
| **Layer 3** | [[抱持]] | 行動制御、保持 | dlPFC、ACC |

### Layer 0: 内受容感覚

呼吸、緊張、心拍のような身体状態の足場に関わる層。意識が最初から抽象的な思考として始まるのではなく、身体の状態を土台に立ち上がることが重視される。Layer 0 自体がすでに問題判断をしているわけではなく、一人称経験の身体的足場を与えている。

[[間主観性]]の研究により、Layer 0 は個人内で完結するのではなく、養育者との相互作用（社会的アロスタシス）を通じて発達することが明らかにされている。

### Layer 1: 予測-誤差ループ

予測と現実の差を照合する層。すべての差がそのまま苦しさになるわけではなく、差の検出と選別が起きている。多くの差は前意識的に吸収され、一部だけが違和感や欠けの芽として残る。

### Layer 2: 生存-信頼評価

その差が何に関わる問題として受け止められるかを方向づける層。生存に関わる問題なのか、信頼や関係に関わる問題なのかによって、情動の向きと行動準備の向きが変わる。[[情動の構成]]が行われる層である。

### Layer 3: 抱持

方向づけられた反応をすぐ固定せず、見直す余地を保つ層。ここがあることで、反応は自動的な短絡だけで終わらず、再評価へ開かれる。[[ネガティブケイパビリティ]]はこの層の認知的条件として位置づけられる。

## 精神分析概念との対応

Bion の精神分析理論を経由して、4層モデルと精神分析概念の仮説的対応が確立されている。

| Bion | 4層モデル | 欠損駆動思考 |
|------|----------|-------------|
| β要素 | Layer 0-1（未処理） | [[欠損]]の前駆体 |
| α機能 | Layer 1→2 への移行プロセス | 欠損化のメカニズム |

| Container | 外的抱持 | 養育者の保持機能 |
| 内在化されたcontainer | Layer 3（抱持） | 自己保持能力 |
| contact-barrier | 4層全体の統合 | 予測モデルの境界 |

## 神経現象学との接続

4層モデルは神経現象学を通して「一人称経験をどの単位で書くか」を見直している。特に重要なのは身体性、差異の前景化、一人称記述単位、前反省的気づきの4点である。

- Layer 0 は身体の背景条件
- Layer 1 は差異の選別入口

Layer 0 と Layer 1 は、意識経験そのものを置き換える層ではなく、一人称経験が立ち上がる前提と入口を与える層である。

## 一人称記述と参照枠

4層モデルの主語は一人称経験である。「何かが足りない」「急に違和感が前に出た」「体が先に緊張していた」といった記述が本論にあたる。

一方、予測誤差、精度重み、脳部位、計算論的な更新といった語彙は、4層モデルを照らすための参照枠であり、それ自体を本論として採るわけではない。

## 精神疾患の4層分類

4層モデルは精神疾患の統一的理解の枠組みを提供する。

| 主要障害 Layer | 疾患群 | 特徴 |
|---------------|--------|------|
| Layer 0 | 身体症状症、心身症、解離 | 内受容感覚の乖離・歪み |
| Layer 1 | 精神病、強迫症 | 予測誤差処理の根本的障害 |
| Layer 2 | 不安障害、うつ病、PTSD | 生存-信頼評価の偏り |
| Layer 3 | 衝動制御障害、ADHD | 抱持機能不全 |
| 複合 | BPD、複雑性PTSD | 複数層にわたる障害 |

## 未確定事項

4層モデルは有力な整理だが、以下の点はなお検討が必要である。

- Layer 1 の差のうちどこからを問題の立ち上がりとみなすか
- Layer 2 に第三の軸が必要か
- Layer 3 を独立した層として扱うべきか、再評価条件とみるか
- 各層を観察可能な記述へどう接続するか
- 前反省的気づきを4層モデルのどこへ返すのが妥当か
